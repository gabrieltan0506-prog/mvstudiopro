# MV Studio Pro - 介面設計文檔

## 設計理念
MV Studio Pro 是一款面向音樂創作者的智能 MV 製作與發布助手。設計風格採用深色系（暗紫/深藍漸變），搭配霓虹色調的強調色，營造專業音樂製作工作室的氛圍。遵循 Apple HIG，確保單手操作流暢。

## 色彩方案
- **Primary**: #8B5CF6 (紫色 - 代表創意與音樂)
- **Secondary**: #EC4899 (粉色 - 代表活力與熱情)
- **Background Light**: #FAFAFA
- **Background Dark**: #0F0D15 (深紫黑)
- **Surface Light**: #F3F0FF
- **Surface Dark**: #1A1625 (暗紫)
- **Foreground Light**: #1C1335
- **Foreground Dark**: #F0ECFF
- **Accent**: #06B6D4 (青色 - 用於數據與分析)

## 螢幕列表

### Tab 1: 首頁 (Home)
- 頂部歡迎橫幅，顯示用戶暱稱和 MV Studio Pro Logo
- 快捷功能入口卡片（3個大卡片）：
  1. MV 分析優化 - 帶音波圖標
  2. 虛擬偶像工坊 - 帶人物圖標
  3. 發布策略 - 帶火箭圖標
- 最近項目列表（帶縮略圖）
- 底部提示：支持的平台圖標（小紅書、B站、抖音、視頻號）

### Tab 2: MV 分析 (Analyze)
- 頂部：上傳區域（支持視頻和音頻上傳）
- 分析結果面板：
  - 技術參數卡片（分辨率、幀率、時長、比特率）
  - AI 分析報告（畫面構圖、色彩分析、節奏匹配度）
  - 改進建議列表（帶優先級標記）
  - 分鏡腳本建議
- 底部操作按鈕：生成優化方案

### Tab 3: 虛擬偶像 (Avatar)
- 頂部：偶像風格選擇器（滑動選擇）
  - 二次元風格
  - 寫實風格
  - 賽博朋克風格
  - 夢幻仙境風格
- 角色配置面板：
  - 性別選擇
  - 外觀特徵（髮型、服裝、配飾）
  - 表情/動作預設
- 批量生成設置：
  - 數量選擇（1-10個角色）
  - 統一主題/獨立風格切換
- 生成結果展示：網格預覽
- 導出選項

### Tab 4: 發布策略 (Publish)
- 平台選擇卡片（四大平台）：
  - 小紅書 (紅色主題)
  - B站 (藍色主題)
  - 抖音 (黑色主題)
  - 視頻號 (綠色主題)
- 每個平台展開後顯示：
  - 5個熱門選題建議（帶預估流量標籤）
  - 最佳發布時間
  - 推薦標籤/話題
  - 封面建議
  - 文案模板
- 一鍵生成發布計劃

## 主要內容與功能

### MV 分析優化
- 上傳 MV 視頻和音頻文件
- AI 自動分析畫面質量、構圖、色彩
- 音畫同步分析
- 生成專業改進建議報告
- 分鏡腳本優化建議

### 虛擬偶像批量製作
- 多種風格模板選擇
- 角色外觀自定義配置
- 批量生成多個虛擬角色
- 角色預覽與管理
- 導出角色資源

### 多平台發布策略
- 針對小紅書、B站、抖音、視頻號的定制化建議
- 5個熱門選題生成
- 最佳發布時間推薦
- 標籤和話題推薦
- 封面和文案模板

## 關鍵用戶流程

### 流程1: MV 分析
用戶點擊「MV 分析」→ 選擇上傳視頻/音頻 → 等待分析（進度條）→ 查看分析報告 → 查看改進建議 → 可選：生成優化方案

### 流程2: 虛擬偶像生成
用戶點擊「虛擬偶像」→ 選擇風格模板 → 配置角色參數 → 設定批量數量 → 開始生成 → 預覽結果 → 導出/保存

### 流程3: 發布策略
用戶點擊「發布策略」→ 選擇目標平台 → 查看5個熱門選題 → 查看發布建議 → 選擇/自定義方案 → 生成發布計劃

## 導航結構
- 4個 Tab：首頁、MV分析、虛擬偶像、發布策略
- 每個 Tab 內使用堆疊導航處理詳情頁
