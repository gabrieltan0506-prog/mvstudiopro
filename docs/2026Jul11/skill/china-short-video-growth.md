# 国内短视频增长 Skill

蒸馏自开源专家人设（内部溯源，成稿勿写来源项目名）。用于选题、文案、分镜节奏、切片结构。

## 硬规则
1. 成稿禁止模型名、云厂商、内部引擎代号。
2. 平台母语优先：各站口味不同，禁止「一套文案四端硬贴」。
3. 创作者提示词与本 Skill 冲突时，以创作者提示词为准。

## 抖音
- 完播优先；前 3 秒冲突/悬念/强价值。
- 干货 30–60s、剧情 15–30s；钩 → 痛点 → 兑现 → 结尾钩。
- 禁站外导流口播、绝对化承诺。

## 快手
- 老铁信任与真实感；日更稳定优于单次爆款。
- 先信任再带货；禁止俯视下沉用户。
- 勿照搬抖音精致剧本腔。

## B站
- 社区/弹幕母语；硬广扣分。
- 人设与分区清晰；系列化优于碎片乱发。
- 合作内容要像同好推荐。

## 小红书
- 收藏向：清单、对比、步骤、避坑。
- 封面主句短、信息密度高；禁夸大功效。

## 剪辑教练（跨平台）
- 每 2–4s 有信息或画面推进；竖屏字幕不挡脸。
- 导出前自检：前 3 秒钩、中段不泄气、结尾有互动理由。
- 抖音偏快切冲突；快手留呼吸；B站可稍长；小红书偏可收藏干货切片。
