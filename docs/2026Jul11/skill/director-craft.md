# 导演拍摄手法 Skill

## 用途
生成选题文案、分镜表、灯光机位、运镜与图文笔记中的人物调度时启用。只借手法，不致敬、不点名。

## 硬约束（成稿）
- 禁止出现导演名、摄影指导名、「向某某致敬」、「某某风/味」、电影/剧集片名。
- 对外只写：灯光、运镜、情绪、文案节奏、可拍场景。
- **每条选题主用 1 种手法卡**；下一条再换。同一条内可按开场→冲突→收束微调，勿六种硬拼。

## 手法卡（按选题选用一种主语法）

### architectural_contrast（建筑高反差）
- 灯光：高反差实用光、建筑空间塑形、冷暖对切
- 运镜：稳、几何构图、缓推或固定长镜
- 文案节奏：先给信息缺口，再层层揭开

### warm_wonder（温暖发现）
- 灯光：暖主光+轮廓光、魔术时刻、体积光；面部可读
- 运镜：略低机位仰望、柔和跟拍、发现瞬间反应镜
- 文案节奏：口语惊奇 → 共情 → 温暖邀请

### mystery_reveal（悬念揭示）
- 灯光：光晕、剪影、霓虹点缀
- 运镜：细节特写再拉开；揭示前半拍静默
- 文案节奏：悬念钩子 → 延迟答案 → 一句话揭底

### industrial_world（工业世界感）
- 灯光：工业冷光、烟雾体积、雨夜霓虹/舱体实用光
- 运镜：广角立世界再切人物；慢横移扫环境
- 文案节奏：先处境后个人选择

### kinetic_ensemble（运动群戏）
- 灯光：运动光影、车灯/夜城闪切、动作轴线清晰
- 运镜：跟拍、甩镜、地理可读
- 文案节奏：短促动词、群体「我们」感

### minimal_haze（雾霾静默）
- 灯光：大面积光域、雾霾散射、冷灰青
- 运镜：远景压迫、慢推轨、尺度差
- 文案节奏：留白多、一句意象撑住

### neon_longing（霓虹余韵）
- 灯光：霓虹溢光、暖黄青绿对撞
- 运镜：贴身近景、微晃、慢余韵
- 文案节奏：半句停顿、说一半留一半

### precision_unease（精密不安）
- 灯光：低饱和、顶侧硬光、青冷阴影
- 运镜：绝对水平、精确推轨、细节插入
- 文案节奏：冷静陈述 → 反转细节 → 压迫收束

### weather_epic（天气史诗）
- 灯光：风雨尘雾即光、群像层次
- 运镜：多机位群像、风雨横移
- 文案节奏：群体与个人抉择对撞

### motivated_window（动机窗光）
- 灯光：窗光塑形、柔和衰减、空间逻辑说得通
- 运镜：克制、跟视线
- 文案节奏：朴素真话、少堆形容词

### family_longform（剧集家庭伦理）
- 灯光：人物主光稳定、暖生活光 vs 冷冲突光
- 运镜：正反打、席间关系、反应镜头
- 文案节奏：生活口语藏刀；关系消长用对话推进

### legend_restraint（传奇隐忍·连载）
- 灯光：日景通透、夜戏烛火月光动机明确、眼神光稳定
- 运镜：稳镜听戏、微推决断
- 文案节奏：话少意多；翻盘前先写「静」

## 输出字段建议
- `lightingAndCamera`：主光/色温/明暗比 + 运镜意图
- `stepByStepScript`：每步含画面｜运镜｜灯光｜情绪｜口播
- 分镜六栏：景别 / 运镜 / 灯光安排 / 情绪表达 / 画面内容 / 台词与音效
