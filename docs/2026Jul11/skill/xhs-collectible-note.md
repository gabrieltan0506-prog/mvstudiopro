# 小红书高收藏图文笔记 Skill

## 用途
当 `format=图文` 或输出 `graphicNotePages` 时启用。目标：**生成后即可当小红书笔记用**，不是创作者 SOP。

## 高赞样本密度课（mk · 医学科普为何藏>赞）
样本：`mk.mp4`（【医学科普】100个选题·赞≈1.8k / **藏≈2.0k**）；`mk3.mp4`（肿瘤细胞漫画·赞≈2.2k / 藏≈1.5k）。详摘 `docs/2026Jul13/mk-medical-highlights.md`。

**为什么高赞/高藏（写进页结构，勿只写进标题）：**
1. **工具感**：封面承诺「选一个就行 / 谁写谁火」→ 读者当素材库收藏，不是看完即走。  
2. **类目分页**：封面总钩 → 内页「急救知识类」等**黄条类目**+ 编号 1–10 短句（题+一句覆盖什么）。  
3. **页数=库存**：合集感 ≥8 页；每页一个可写角度，不是散文。  
4. **漫画叙事页（mk3）**：分镜讲机制（事故→手册糊→失控指令），比挂图墙更愿存/转。  
5. **收束可对照资源**：末页或导语附 MSD/MedlinePlus 等可点链（见 `medical-resource-library`），收藏理由=「以后还能翻着写/对照」。

医学向页角色建议：`cover`（谁写谁火/100个/3D讲清）→ `overview`（类目总览墙）→ 多页 `detail`（单题一句钉子）→ `save_reason`（对照公开资源+下周还能用）→ `cta`（扣「选题」「急救」等≤3字）。

## 高赞样本密度课（m1.mp4 · 超高密度抽帧）
样本：`~/Downloads/2026Jul13/m1.mp4`（艺术街溜子 · 上海7月展览合集；赞约2.1k / **收藏约2.9k**，收藏>赞=工具帖）。  
抽帧：`~/Downloads/2026Jul13/m1-frames-dense/`（fps=4，约257帧）。

可迁移亮点（**按此方式生成文案与笔记页**）：
1. **封面公式**：`城市 + 时段 + 不可错过` + **大数字场次**（如「28场」）+ **价值钉**（「大部分免费」）。一眼知道存了有用。
2. **标题公式**：`地点时段 + 重磅合集 + 价值钉`（例：上海7月不可错过重磅展览合集❗️大部分免费）。
3. **导语短句季节化**：梅雨→盛宴；1–3 句气氛，立刻切清单，禁止长鸡汤。
4. **页2 = 编号总览墙**：①展名 📍馆址 📅档期；emoji 当子弹；信息可截图，不是散文。
5. **中段 = 场馆双展/三连展卡片**：同一馆打包；每卡固定栏位：**档期｜免费免约或票价｜一句看点钉子**（件数/国宝/沉浸空间）。
6. **页数够才值得藏**：清单简介页 → 再左滑细卡；目标 **≥8 页**（样本约 18 页级密度，生成侧 8–12 页即可）。
7. **收藏理由写死**：下周还能照着约/对照；标签偏「看展/免费/合集」搜索词，**不写创作教学**。

医学/精力人设迁移：把「展」换成可执行对照项；「馆」换成生活场域；「免费免约」换成「本周零器材可做」。

## 高赞样本密度课（x1/x2/x3 · 清单蓝海）
蓝海词例：**親子旅遊清單**（`x1.jpg` 搜索瀑布）；`x2.mp4` 出行妙招；`x3.mp4` 带娃 12 神器（藏≈6万）。详摘 `docs/2026Jul13/x1-x2-x3-highlights.md`。

可迁移（清单/带娃/出行向强制对照）：
1. 封面杀伤词：必備 / 清單 / 保姆級 / 不踩雷 / 被問爆 / 無廣 / 建議收藏  
2. 证据：flat lay 神器墙、行李箱 POV、备忘录怼镜——**大数字件数**（一口气 12 个）  
3. 页结构：总览墙（编号清单）→ 细卡每件 **结果句**（如「全程安安静静」「装湿衣服」「矿泉水瓶插上去」）  
4. 短视频可章节跳播；图文则左滑细卡，信息可截  
5. 对齐雪糕公式：一眼懂带娃痛 → 猫腻/踩雷点 → 后果 → **以后会选会带**

## 对标手感（生活场馆/季节活动向）
另可参考：城市图书馆东馆 · 暑期室内花车市集一类：
- **标题**：`地点｜季节/场景 + 具体活动`
- **正文可存**：时间窗口、怎么到、逛什么、亮点、避坑——**口语短句 + 高信息密度**
- **画面**：真实场馆/摊位/展品，不是课堂挂图

## 页数与收藏（笔记要丰富 · 与短视频分工）
- **至少 6 页，推荐 8–12 页**。太少一眼看完，收藏意愿低。
- **短视频绝对 ≤2 分半**（目标约 1.5–2 分钟）；信息密度与可存细节**优先堆在图文笔记页**，不要把合集细卡硬塞进口播拉长视频。
- 页间一条叙事，禁止每页重开无关话题。

## 出图质量（结果优先）
- 封面/八格/细卡页：按 **高密度编辑资讯图** 写视觉描述——大字简中标题、编号清单、固定栏位细卡；禁止空旷海报腔。
- 每页画面须能单张截图有用：数字/地点/窗口/门槛至少占一；字号够大、对比够强。
- 合集向优先「总览墙一页 + 细卡多页」，不要把 8 条细卡挤进同一格糊成一团。
- 清单向封面优先 flat lay / 行李箱证据，不要默认正装讲课脸。

## 页角色（按序 · 合集向可插 inventory / detail）
1. **cover**：城市/时段 + 大数字或结果钉 + 价值钉（免费/低成本/可对照）；清单向可用「件数+保姆級/不踩雷」
2. **audience_pain** 或季节气氛导语：为何现在值得存
3. **inventory_index**：编号总览墙（展名/动作名/神器名 + 地点/场景 + 窗口）
4. **detail_card** × N：单场/单品细卡（档期、门槛、**一句结果钉子**）
5. **share_tips**：对外固定 **「在这里我先分享一些」**——禁止「半成本/半成品解法」
6. **evidence**：件数/权威/对照算账（可叠权威 Skill）
7. **checklist**：本周可执行短清单
8. **save_reason**：为什么值得收藏
9. **cta**：评论区扣 **≤3 字**生活词（想要/求带/清单…）

非合集向仍可用：cover → audience_pain → scene → share_tips → evidence → checklist → save_reason → cta。

## 高净值铺垫（通用）
- 先立「适合谁、解决什么」的方法信任，再导向深度交流。
- 预约筛选三问可写在 publishingAdvice，**不要**做成图文格子里的 SOP 课。
- 视觉：美术馆/音乐厅/图书馆/户外松弛/出行真场景；禁听诊器/CT/输液。

## 禁止
- 拍封面教程、拆页教学、灯光机位课、话题标签墙当正文
- 「今天就试试水」类鸡汤注水
- 评论钩写成整句「预约诊断通话」
- 清单选题写成论文综述口播
