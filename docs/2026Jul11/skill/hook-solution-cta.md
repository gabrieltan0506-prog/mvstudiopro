# 钩子·分享要点·咨询闭环 Skill

## 用途
所有平台选题文案、自定义选题、决策智库扩写、选题初选默认启用。让用户停滑并想继续问你，而不是纯百科。

## 对外话术（硬）
- 中段解法对外**必须**写成：**「在这里我先分享一些」** + 2–3 个可立刻用的要点。  
- **禁止**对观众说「半成本解法」「半成品解法」——刺耳、像在卖关子羞辱人。  
- 完整方案留深度交流 / 顾问路径，但语气是分享而非刁难。

## 评论钩子（硬 · ≤3 字）
- `commentHooks` / 评论关键词：**每个不超过 3 个汉字**，生活化。  
- 推荐词表：想要、求带、慢生活、优雅、收藏、清单、同款、求链、学到了、共鸣。  
- **禁止**整句：「预约诊断通话」「私信我领完整方案」「点击主页咨询」。长引导可放正文末页，但评论扣词必须短。

## 高度需求结构
1. **先内化**：客户是谁、核心痛点 1–2 个、分享价值、爽点。  
2. **标题**：好奇缺口 / 反常识 / 反差 / 时事；忌正确但无聊。  
3. **钩子（黄金三秒）**：痛点火花 + 爽点苗头。  
4. **正文**：开场钩子 → 中段「在这里我先分享一些」→ 结尾收满 + 评论短钩 CTA。  
5. **分镜**：前 3 秒痛点+爽点苗头同框或紧挨。

## 痛点 / 爽点
| 阶段 | 痛点 | 爽点 |
|------|------|------|
| 黄金三秒 | 点名窘境 | 痛快苗头 |
| 中段 | 代价与误区 | 分享要点逐条兑现，仍留缺口 |
| 结尾 | 卡点收束 | 爽点说满 + 短评论钩 |

## 数目
- 分享要点优先 **2–3 个**，最多 5 个。  
- 数字钩子可拉完播（先抛最反常识的一条）。

## 文化人物
- 可与 `cultural-diversity` 叠用；**同人默认全批最多 1 条**（代码侧 dedupeKey 硬去重）。

## 强监管
- 叠 `review-safe-voice`（强监管优化表达）。
