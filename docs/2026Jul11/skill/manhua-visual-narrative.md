# 漫剧视觉叙事 Skill

蒸馏自开源「静帧提示 / 叙事设计 / 视觉叙事」人设。用于漫剧大纲、分镜表、关键静帧提示词。

## 硬规则
1. 成稿禁止模型名、供应商名；手法用中性标签，不点名致敬。
2. 以工作台「【分镜 N·静帧】」动作为准，示范图只借空间与物件。
3. 创作者提示词优先于本 Skill。

## 静帧提示
- 分层：身份气质 → 服饰时代 → 动作关系 → 场景纵深 → 光色 → 景别 → 画风硬锁。
- **CG 漫剧**：二次元/厚涂插画，禁止仿真人皮肤与街拍。
- **仿真人**：电影剧照，禁止塑料感与画面烧字。
- **古装**：发髻袍服甲胄齐全；禁西装 T 恤球拍手机霓虹。
- 画面零字幕、零水印；对白只作表演。

## 叙事分镜
- 每镜戏剧功能：揭示 / 施压 / 关系 / 兑现。
- 关系镜须同框；禁止单人肖像糊弄对峙/递接。
- 输出建议含「## 故事大纲」与「## 分镜表」（镜号、时长、运镜、动作、情绪）。

## 视觉弧
- 开场钩 → 中段升温 → 末镜悬念；跨镜服装/光向/道具连续。
- 首镜须一眼冲突，禁止平淡空镜开场。
