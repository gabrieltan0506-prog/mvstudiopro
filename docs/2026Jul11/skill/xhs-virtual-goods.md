# 小红书虚拟资料店 Skill

## 用途
选题、笔记文案、封面主图、商品卡话术涉及**虚拟资料 / 数字产品 / 模板包 / 题库**时启用。  
样本：`~/Downloads/2026Jul16/1.png`–`10.png`、`d1.png`/`d2.png`（OCR 归档见同日扫描纪要）。

## 卖的是什么（不是空气）
把学习/工作/生活中**反复需要**的信息，整理成**可在线交付**的数字产品：替用户完成搜集、筛选、核对、分类、排版，卖的是**节省时间的可执行方案**。

常见品类：竞赛计划书与路演模板、家长会发言稿/流程、周月报 PPT、表格工具与清单、授权软件/素材服务、职业场景资料包（备课/合同/运营 SOP）等。

记住三件事：**选品看需求 · 内容有价值 · 合规是底线**。

## 选品与产品矩阵
1. 先定**一个细分场景**（宁窄勿宽）：明确人群 + 一组相关问题。  
2. 产品线递进：入门（低决策门槛）→ 核心（主利润）→ 进阶（复习/系统提升）。例：古诗词默写 → 阅读理解 → 单元考点 → 期末综合。  
3. 低价不能代替质量；须原创或合法授权，描述/交付/售后说明清楚。

## 搜索与标题
资料有强搜索属性。标题公式：  
`适用人群/年级 + 科目 + 具体需求 + 资料类型`  
例：「三年级语文古诗词默写」「六年级阅读理解专项」。  
禁止堆砌无关热词；关键词须真会被搜。

## 时间差与低粉爆款
- **时间差截流**：考季/春招等提前 0.5–1 月布局关键词。  
- **低粉爆款挖掘**：类目词 + 粉丝少于 500 + 近 15 天高赞，验证新兴需求。

## 笔记结构（直白优先于精美）
1. 开头：真实困扰（场景+紧迫），不先推货。  
2. 中段：适合谁 / 含什么 / 怎么用；可 5–9 张核心截图拼接。  
3. 文案三句：痛点价值 → 人群过滤 → 交付物（如「填空版+范文」）。  
4. 无需真人出镜；禁「保证回本/官方包开」等诱导。

## 主图 = 痛点收割机
- 大字标明商品内容与结果（如「刷完这 500 题，阅读理解拿满分」）。  
- 禁模糊截图；可加真实反馈截图、收发货/交付方式建立信任。  
- 与 `cover-stop-scroll` 联动：少字、提亮关键字、结果钉。

## 合规（硬）
- 文字/图片/字体/音乐商用授权；模板可否转售。  
- 不搬运他人课程/付费资料/个人信息；保留授权凭证。  
- 不夸大收益；面向学生家长注意版本准确性与未成年人规则。  
- 虚拟可重复交付 ≠ 零成本：仍有制作、设计、平台费、售后、更新成本。

## 执行十步（可复述进选题 conveyGoal）
定人群 → 收真实需求 → 核版权类目 → 设计矩阵 → 原创内容 → 预览材料 → 布局搜索词 → 持续笔记 → 记数据 → 按反馈迭代。
