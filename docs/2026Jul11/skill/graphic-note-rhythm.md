# 图文笔记版式节奏 Skill

## 用途
**仅当 format=图文**（含小红书双页/3×4 笔记）启用本约束。短视频分镜**不要**套用本 Skill 的格规则。

## 体裁·硬约束（最重要）
图文笔记是给**粉丝直接收藏转发**的生活/知识帖，不是给创作者看的「技术指导手册」。

**禁止任何格子出现：**
- 如何拍封面 / 今晚拍封面素材
- 如何拆成八页、十二格、页码清单
- 同步录 60 秒 / 怎么拍怎么发
- 发布建议、话题标签墙、分镜口播、灯光机位
- 落地执行三步曲、内容生产 SOP

**对标正常可发笔记结构（第二张样本风）：**
封面钩子 → 你是谁/痛点 → 误区 → 场景/关系/节律要点 → 常见问 → 评论领清单

末几格只能是互动/清单 CTA，**禁止**变成「拍素材 / 拆页 / 录视频」教学。

## 版式硬约束
1. 格间是**一条叙事**：痛点 →「在这里我先分享一些」要点 → 生活场景印证 → CTA；禁止每格重开一个无关话题。合集向可插总览墙+细卡（见 `xhs-collectible-note` m1 密度课）。
2. 信息密度：每格一个主信息 + 短句；禁止单格长段落墙。
3. 3×4 十二格：前 1–2 格钩子/痛点，中段场景/关系/节律要点，末 1–2 格互动或清单引导（非创作教学）。
4. 双页/八格：封面格只承载封面主句级信息；内页再展开。
5. 视觉明快、生活化；少暗调说明书风。

## 与平台
小红书图文是流量重要形态；**全案默认至少约一半选题为图文**（见 `platform-native` 配额），便于连发 2–3 条笔记测流。同题可同时保留 B站/视频号短视频变体，**不冲突**。
