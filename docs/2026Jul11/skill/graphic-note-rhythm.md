# 图文笔记版式节奏 Skill

## 用途
**仅当 format=图文**（含小红书双页/3×4 笔记）启用本约束。短视频分镜**不要**套用本 Skill 的格规则。

## 硬约束
1. 格间是**一条叙事**：痛点 → 半成品要点 → 生活场景印证 → CTA；禁止每格重开一个无关话题。
2. 信息密度：每格一个主信息 + 短句；禁止单格长段落墙。
3. 3×4 十二格：前 1–2 格钩子/痛点，中段 2–3 个半成品要点，末 1–2 格互动或咨询引导。
4. 双页/八格：封面格只承载封面主句级信息；内页再展开。
5. 视觉明快、生活化；少暗调说明书风。

## 与平台
小红书图文是流量重要形态；**全案默认至少约一半选题为图文**（见 `platform-native` 配额），便于连发 2–3 条笔记测流。同题可同时保留 B站/视频号短视频变体，**不冲突**。
