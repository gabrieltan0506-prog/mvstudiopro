# 选题文案去无聊 Skill

## 用途
全案 Stage2、自定义选题、决策智库扩写**默认启用**。管的是：**文案怎么说才不无聊、不说教**，以及**前 3 秒能不能留住人**。  
封面表情/夸张姿势细节同时遵守 `cover-stop-scroll`；咨询闭环仍遵守 `hook-solution-cta`。

## 硬约束·口吻（幽默风趣 ≠ 低级梗）
1. **像朋友吐槽，不像老师训话**：可用反问、自嘲、夸张比喻、轻微抬杠；**禁止**「你应该」「必须明白」「请务必」说教腔，禁止正确但催眠的长段玄学。
2. **呈现形式要有戏**：短句、对白感、物件出镜；可穿插「内心 OS / 反转一句 / 假装正经再拆台」；禁止论文摘要腔（「综上所述」「在此基础上」「具有重要意义」）。
3. **标题必须有戏**：具体物件 / 身体动作 / 反差 / 不信感至少占一；禁止空心大词（「古籍生命观」「高知状态管理」单独做标题且无场景）。
4. **六条辨识度**：场景、情绪、开头句式明显不同；禁止六条同一复读机。

## 硬约束·结构（痛点 + 爽点前置 · 黄金三秒）
为高完播与停滑，**前 3 秒 / 第一屏**必须同时出现：

| 前置 | 要什么 | 反例 |
|------|--------|------|
| **痛点前置** | 一眼戳中「就是我」的窘境、误区或被忽视的代价 | 先讲定义/背景/「今天我们来聊聊」 |
| **爽点前置** | 立刻给一点痛快：反常识结论、解气对比、可感知的「原来如此/终于有人说了」、半成品爽感预告 | 全程只骂痛不给出口；或把爽点藏到文末 |

推荐开场顺序（可口语微调，勿颠倒成百科）：
1. **0–3 秒**：痛点一击 + 爽点火花（一句反转/解气/不信感）  
2. **接着**：点名这类人 + 展开痛点画面  
3. **中段**：2–3 个半成品要点（留白完整方案）；清单最多 5 点；可用「五个妙招先说第三个」拉完播  
4. **收束**：关系型 CTA（评论/私信），仍可带一点幽默，勿突然变合规宣读

`copywriting` / `hook` / 短视频前 3 秒口播、图文封面+图2 **都必须**体现「痛点与爽点前置」；不要把爽点当成结尾彩蛋。

## 硬约束·完播与画面提示（给分镜/封面的台词）
1. **黄金三秒可拍**：第一镜须有表情或动作突变（错愕、坏笑、瞪眼、失衡、发球定格等），口播与画面同拍痛点+爽点，禁止「端坐开场白」。  
2. **表情多元、姿势可夸张**：为完播服务——瞪大眼、坏笑、自嘲捂脸、探身怼镜头、差点摔倒、网球发球展体等轮换；同批勿同一张「上课脸」。细目见 `cover-stop-scroll`。  
3. **中段每 80–120 字**至少一个具体道具或动作；至少一次小意外/态度反转，降低中途划走。

## 无聊 / 说教反例（禁止复刻）
- 「首先我们从生命节律谈起……」  
- 「周末躺两天还像没充电」后接大段庄子论文。  
- 「用生命科学与古籍观察现代人」当钩子，无物件、无痛点、无爽点。  
- 通篇「你要学会爱自己」鸡汤训话，无画面无反击。

## 有趣正例方向（勿照抄）
- 痛点+爽点同句：「合上电脑那一秒，你的身体其实还在开会——但它比你的老板诚实。」  
- 「黑胶转完了你还不抬唱针——休息也是。」  
- 「全公司找不出一个会休息的人？」  
- 「五个妙招——先说第三个」（数字钩子拉完播）  
- 影视桥段：「藏海传高压局里养精力的 3 个动作」（半成品，非剧透墙）  
- 反差新创：「博士毕业去端盘子，第一天就懂了××」  
- 器官拟人：「我在你胸腔里值夜班，你却还在刷消息」（配合 `crossover-popsci`）  
- 运动线：「发球那一下，心肺比你的 KPI 更诚实。」

## 自检
- [ ] 朋友看标题会不会回「所以呢？」——会，就重写。  
- [ ] 前 3 秒 / 第一屏有没有**同时**碰到痛点与爽点？  
- [ ] 通篇像不像在训人？有没有一句可被当成「老师训话」的？有则改成吐槽或对照。  
- [ ] 半成品是否仍留白（生动 ≠ 写尽咨询方案）？

## 职责边界
- **本文**：文案口吻、节奏、痛爽前置、反说教。  
- **`cover-stop-scroll`**：封面少字 + 表情姿势夸张/多元。  
- **`hook-solution-cta`**：客户→痛点→半成品→咨询闭环（本 Skill 要求把痛点/爽点**提前到开场**）。  
- **`contrast-reversal-climax`**：身份反差与情绪反转弧（可叠用加强完播）。
