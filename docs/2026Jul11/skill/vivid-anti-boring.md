# 选题文案去无聊 Skill

## 用途
全案 Stage2、自定义选题、决策智库扩写**默认启用**。管的是：**文案怎么说才不无聊、不说教**，以及**前 3 秒能不能留住人**。  
封面表情/夸张姿势细节同时遵守 `cover-stop-scroll`；咨询闭环仍遵守 `hook-solution-cta`；跨界科普叠用 `crossover-popsci`。

## 最高优先级·禁止再出「干巴巴知识课」
下列写法 **一律判失败并重写**（808 批次即为反面教材）：
- 标题/文案只绕着「恢复系统 / 节律 / 留白 / 自律 / 爵士隐喻 / 《内经》金句」打转，**没有一个具体的人的故事**。
- 通篇「概念 → 三点清单 → CTA」，像顾问 PPT，不像会转发的故事。
- 六条选题只是同一套「高压中年缺恢复」换皮（咖啡/爵士/庄子/内经轮换）。
- **知识人人都听过**；你只堆知识点 = 没人想听。

## 硬约束·个案故事（每条必有）
1. **每条** `title` / `hook` / `copywriting` / `detailedScript` **必须**嵌入至少 **1 个生动个案**（可含真实公共事件气质的匿名化故事，或可拍的「某人」切片），**禁止**只讲抽象知识。
2. **案例优先于知识点**：先让人「看见一个人出了什么事」，再落 1–2 个机制点；知识是配菜，故事是主菜。
3. **案例类型菜单**（每条至少选一类；同批六条须轮换，勿六条都写猝死）：
   - **猝不及防**：前几天还炫耀身体素质好，几天后因突发状况离世/倒下（不渲染血腥，强调「以为自己没事」的反差）。
   - **年轻却中招**：如年轻女孩确诊渐冻症、同龄人突发心脑血管事件——**可包括但不限于**；目的是打破「还年轻所以没事」。
   - **身边对照**：同事/父母/球友「看起来很自律却垮掉」的具体一周切片（咖啡毫升、睡眠小时、最后一次说「我没事」）。
   - **反差身份**：博士/高管/运动员也会「欠费」；或饭桌劝父母失败的具体一句对话。
   - **公共记忆型**：可用广为人知的公共叙事气质（不捏造可指名的私密医疗档案；匿名+观察向）。
4. **标题也要有故事钩**：优先「人名/身份 + 意外转折」或「具体场景 + 出事感」；禁止空心大词标题（「古籍生命观」「高知状态管理」）。
5. **审核友好边界**（与 `review-safe-voice` 同开时）：故事用于**唤醒与对照**，不诊断观众、不开药方、不承诺疗效、不恐吓「你一定会得××」；用「观察/听说/身边/公开报道气质」表述。

## 硬约束·口吻（幽默风趣 ≠ 低级梗）
1. **像朋友讲一件真事**，不像老师训话；可用反问、自嘲、夸张比喻；**禁止**「你应该」「必须明白」「请务必」。
2. **呈现形式要有戏**：短句、对白、物件出镜；可穿插「内心 OS / 反转一句」；禁止论文摘要腔。
3. **六条辨识度**：场景、情绪、开头句式、**个案主角**明显不同。

## 硬约束·结构（痛点 + 爽点前置 · 黄金三秒）
| 前置 | 要什么 | 反例 |
|------|--------|------|
| **痛点前置** | 前 3 秒先甩个案火花或「以为没事」反差 | 先讲定义/「今天我们来聊聊恢复系统」 |
| **爽点前置** | 立刻给痛快：原来如此 / 解气对照 / 半成品预告 | 全程只恐吓不给出口；或爽点藏文末 |

推荐开场：
1. **0–3 秒**：个案一击（或「他还在说身体好」）+ 爽点火花  
2. **接着**：点名这类人 + 把故事讲满画面  
3. **中段**：故事里长出 2–3 个半成品要点（最多 5）；可用「五个妙招先说第三个」  
4. **收束**：关系型 CTA；勿突然变合规宣读

## 有趣正例方向（勿照抄句式，照抄结构）
- 「前几天还在饭局上说自己身体素质好的人，这周突然就不在了——他缺的不是鸡血，是早就该听见的警报。」  
- 「二十多岁的姑娘被确诊渐冻症的那天，朋友圈还在发健身房自拍；你还好意思说『还早』？」  
- 「合上电脑那一秒，你的身体其实还在开会——但我先讲完隔壁老王的故事。」  
- 器官拟人仍可用，但**拟人句后面必须接一个真人个案**，不能只剩隐喻。

## 职责边界
- **本文**：口吻、去无聊、**强制个案故事**、痛爽前置。  
- **`crossover-popsci`**：跨界科普叙事弧；须与本 Skill 的个案硬约束叠用。  
- **`cover-stop-scroll` / `hook-solution-cta` / `contrast-reversal-climax`**：封面、咨询闭环、情绪反转。
