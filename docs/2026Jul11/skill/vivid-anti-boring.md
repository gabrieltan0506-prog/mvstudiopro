# 选题文案去无聊 Skill

## 用途
全案 Stage2、自定义选题、决策智库扩写**默认启用**。  
封面细节见 `cover-stop-scroll`；咨询闭环见 `hook-solution-cta`；跨界科普见 `crossover-popsci`；场域配额见 `lifestyle-diversity`。

## 硬门槛·感情与共鸣（每条都要 · 无例外）
**所有** `title` / `hook` / `copywriting` / `detailedScript` 必须让人**有感觉、能共鸣**——像朋友在说话，有情绪温度、有「就是我」或「心疼一下」的触点。  
这是**必要项**。干巴巴、正确但催眠、顾问 PPT 腔 → **一律判失败重写**。

共鸣可以来自：物件尴尬、关系张力、拟人委屈、生活对照、自我吐槽、反差、热播桥段、**或**（见下）真人案例——**不要求每条都用同一种方式**。

## 硬门槛·「雪糕公式」可看结构（每条要对齐 · 无论文腔）
观众要的是**一眼懂、两记痛、以后会选**，不是把论文结果念给他们听。

**金牌样本（雪糕）——按此结构迁移到任意题材：**
1. **一眼懂的生活局**：原来不能给孩子一直吃雪糕（父母/照顾者立刻代入，不是「添加糖代谢机制概述」）。
2. **痛点 1（猫腻/反常识）**：配料表原来有猫腻（能翻、能指、能截图）。
3. **痛点 2（后果钉子）**：孩子食欲下降、影响发育（说到心疼，不诊疗恐吓）。
4. **收束能力感**：以后懂了怎么选雪糕（观众带走**可执行选择**，不是「请遵循指南原则」）。

| 合格（愿看） | 失败（像上课） |
|------|--------|
| 不能天天给娃雪糕：配料猫腻 + 食欲掉了 | 读《膳食指南》领悟儿童添加糖管理三原则 |
| 翻背面前三名，你以为是奶其实是糖 | 哈佛实验室揭秘糖代谢与下丘脑食欲调控 |
| 看完会选：糖占比 / 配料排序 / 少买哪类 | 用××实验观察慢病风险的 N 招 |

**每条自检（不过就重写）：**
- [ ] 开场是不是**生活局**，而不是「研究结论/机制名词墙」？
- [ ] 有没有 **≥2 个痛点钉子**（猫腻/反常识 + 后果/体感）？
- [ ] 结尾有没有「**以后会选/会做**」的能力感？
- [ ] 把机制名词全删掉，普通人还看不看得懂？删完看不懂 → 失败。

## 硬门槛·禁止「读论文结果」式选题（b2 反面教材）
下列标题结构 **直接判失败**，即使文案有感情也要重写切口：
- `读《××》领悟 / 留下 / 藏着…方案`
- `哈佛实验室揭秘 / 针对…的 N 招…法`
- `用××观 / ××管理术 / ××实验` 观察慢病（无可拍生活场面）
- 标题像会议摘要：堆 `皮质醇、线粒体、HRV、内皮、迷走神经`，观众第一反应是「又在上课」
- 通篇像把论文摘要「翻译成人话」但仍在上课（有机制无生活局、无痛点对、无能感收束）

**合格方向**：先给**人正在经历的局**（追剧、宵夜、加班、劝父母、孩子要雪糕），再轻轻点机制；热播高压剧（如《藏海传》）优于再读一遍苏轼/史记。

| 想看 | 不想看 |
|------|--------|
| 别被《藏海传》权谋骗了：高压局里先保命的身体动作 | 读《史记》…英雄史观抗压实验 |
| 凌晨路边摊：先别戒宵夜 | 5 个迹象预警血管碳化·哈佛实验室 |
| 听完这段爵士你为什么更想躺？ | 苏东坡冷灶哲学与现代血管弹性 |
| 雪糕：不能一直给娃吃 → 配料猫腻 → 食欲/发育 → 以后会选 | 儿童添加糖摄入与发育风险的循证要点 |

## 案例：看情况加入（不是每条强制）
| 选题类型 | 怎么做 |
|----------|--------|
| **疾病 / 慢病风险 / 误区纠正** | **建议**加故事与共鸣切片——知识人人听过，故事才钉得住 |
| **器官功能 / 机制科普** | 拟人、尺度具象、生活对照即可；**不必**硬塞真人病例 |
| **生活节律 / 审美 / 关系沟通** | 场景、对白、物件出戏；清单可用 |
| **畅销品 / 配料 / 儿童零食** | 强制对齐「雪糕公式」+ `4season-fmcg-popsci` / `label-debunk-copy` |

**清单没有禁止**：2–3 点半成品、最多 5 点完全可以；讨厌的是「只有清单、没有感情」。

## 禁止再出的「干货课」（808 / b2 / 苏轼复读反面）
- 六条同质复读「恢复系统 / 节律 / 留白 / 自律 / 爵士隐喻 / 《内经》金句 / 苏东坡冷灶」。
- 通篇正确但催眠，像顾问 PPT 或文献综述口播——**观众刷短视频是来放松的，不是来听课**。
- 讲疾病知识点时零共鸣零故事，只剩说明书。
- 同人换皮：条条苏轼，或同观点换标题。

### 现时长硬限（短视频）
- **目标成片：约 1 分半～2 分钟**（建议时间轴落在 **[00:00–01:30]～[00:00–02:00]**）。
- **绝对上限：≤ 2 分半（≤02:30）**——超时即失败重写；将心比心，再长观众也看不下去。
- **禁止**为凑字数把口播写成接近或超过 2 分半；钉子密、条数少，宁可短不要注水。
- 口播总量大约 **220–380 字**口语即可（顶格也勿为撑满 2:30 注水）；字幕 **8–12 句钉子**封顶。
- **图文笔记另走 `xhs-collectible-note`**：笔记要丰富（8–12 页），**不要**用短视频的短篇幅去砍图文。

可迁移亮点（`format=短视频` 或视频向 `detailedScript` 时强制对照）：
1. **标题杀伤词**：`封神 / 真·天花板 / 才是…` + **只推 3 个**（精选感，不做 28 条念经）。
2. **开场字幕极短**：如「不是小展」「是整个上海都在期待的文博天花板」——一句一个钉子。
3. **稀缺倒计时**：展期虽长，但「T形帛画只展45天」「想看原件趁早冲」。
4. **量感钉子**：472件 / 近3000件 / 4米石碑——数字可截，每条视频 **3 个量感钉封顶**。
5. **反差口语**：感觉要把某馆「宝贝都搬空了」；「这些你平时肯定看不到」。
6. **收束低成本 CTA**：一句「低成本首选」+ 评论钩 ≤3 字，**10 秒内收束**。
7. **运镜**：洁净标题卡 → 场馆大场面 → 展品特写；字幕居中大字，禁止说明书墙。

图文合集密度见 `xhs-collectible-note`（m1）；本处只管**讲解短视频节奏与时长**。

## 硬约束·口吻与结构
1. 像朋友说话；禁「你应该」「必须明白」训话。  
2. 标题要有戏：物件 / 动作 / 反差 / 不信感 / 热播局至少占一。  
3. 六条辨识度：场景、情绪、开头句式明显不同；典籍主容器 ≤2（见 `cultural-diversity`）。  
4. 结构对齐雪糕公式：生活局 → 痛点1 → 痛点2 → 以后会选；前 3 秒痛点火花 + 爽点苗头；中段「在这里我先分享一些」；结尾收满 + CTA。

## 自检
