# 选题文案去无聊 Skill

## 用途
全案 Stage2、自定义选题、决策智库扩写**默认启用**。  
封面细节见 `cover-stop-scroll`；咨询闭环见 `hook-solution-cta`；跨界科普见 `crossover-popsci`。

## 硬门槛·感情与共鸣（每条都要 · 无例外）
**所有** `title` / `hook` / `copywriting` / `detailedScript` 必须让人**有感觉、能共鸣**——像朋友在说话，有情绪温度、有「就是我」或「心疼一下」的触点。  
这是**必要项**。干巴巴、正确但催眠、顾问 PPT 腔 → **一律判失败重写**。

共鸣可以来自：物件尴尬、关系张力、拟人委屈、生活对照、自我吐槽、反差、**或**（见下）真人案例——**不要求每条都用同一种方式**。

## 案例：看情况加入（不是每条强制）
| 选题类型 | 怎么做 |
|----------|--------|
| **疾病 / 慢病风险 / 误区纠正** | **建议**加故事与共鸣切片（猝不及防、年轻中招如渐冻症等可含但不限、身边对照）——知识人人听过，故事才钉得住 |
| **器官功能 / 机制科普** | 拟人、尺度具象、生活对照即可（「谢谢你，我的心脏」弧）；**不必**硬塞真人病例 |
| **生活节律 / 审美 / 关系沟通** | 场景、对白、物件出戏；清单可用 |

**清单没有禁止**：2–3 点半成品、最多 5 点完全可以；讨厌的是「只有清单、没有感情」。

## 禁止再出的「干货课」（808 式反面）
- 六条同质复读「恢复系统 / 节律 / 留白 / 自律 / 爵士隐喻 / 《内经》金句」，全无感情温度。
- 通篇正确但催眠，像顾问 PPT。
- 讲疾病知识点时零共鸣零故事，只剩说明书。

## 硬约束·口吻与结构
1. 像朋友说话；禁「你应该」「必须明白」训话。  
2. 标题要有戏：物件 / 动作 / 反差 / 不信感至少占一。  
3. 六条辨识度：场景、情绪、开头句式明显不同。  
4. 前 3 秒：痛点火花 + 爽点苗头（火花可以是情绪触点，不必须是病例）；中段半成品；结尾收满 + CTA。

## 自检
- [ ] **这条有没有感情与共鸣点？**（必要）  
- [ ] 疾病向：要不要加一个故事钉子？（酌情）  
- [ ] 器官向：有没有拟人或生活对照？（不要求病例）  
- [ ] 朋友会不会回「所以呢，知识点我都知道」？
