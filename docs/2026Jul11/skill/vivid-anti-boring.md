# 选题文案去无聊 Skill

## 用途
全案 Stage2、自定义选题、决策智库扩写**默认启用**。没人看的内容再「正确」「高级」也没用。目标：停滑、想点开、想看完、想评论。

## 硬约束
1. **标题必须有戏**：具体物件 / 身体动作 / 反差 / 不信感 至少占一；禁止空心大词堆叠（例：「古籍生命观」「高知状态管理」单独做标题且无场景）。
2. **钩子必须口语可出镜**：像对人说话，1–2 句内出现可拍画面；禁止「首先我们从××谈起」。
3. **正文节奏**：短句为主；每 80–120 字至少一个**具体道具或动作**；中段要有小意外或反转；禁止论文摘要腔（「综上所述」「在此基础上」「具有重要意义」）。
4. **六条辨识度**：场景、情绪、开头句式明显不同；禁止六条同一「典籍金句→生命科学→三个观察→评论区留词」复读机。
5. **半成品仍要留白**：生动 ≠ 把咨询方案写尽；仍遵守 `hook-solution-cta`。
6. **自检一句**：若把标题发给朋友，对方会不会回「所以呢？」——会，就重写。

## 无聊反例（禁止复刻气质）
- 「周末躺两天还像没充电：真正偷走精力的，是你没做这个收尾动作」后接大段庄子论文。
- 「用生命科学与古籍观察现代人的节律」当钩子，无物件、无动作、无情绪。
- 全段正确建议，读完不想点赞也不想评论。

## 有趣正例方向（勿照抄）
- 「合上电脑那一秒，你的身体其实还在开会」  
- 「黑胶转完了你还不抬唱针——休息也是」  
- 「全公司找不出一个会休息的人？」  
- 器官拟人：「我在你胸腔里值夜班，你却还在刷消息」（配合 `crossover-organ-popsci`）

## 与封面
封面只吃短 `coverHeadline`；正文再展开。不要把有趣全写进封面字墙。
