# 跨界科普叙事 Skill

## 用途
把「课堂说明书」改成**有情绪、有个案、愿收藏转发**的科普。  
参考样本气质（`~/Downloads/2026Jul12/1.mp4` · 「谢谢你，我的心脏」类）：**主体拟人** + **生活对照** + **真人故事钉子**。  
**器官不是唯一题材**；同一批选题勿六条都写心脏，也**勿六条都只剩爵士/内经隐喻而无个案**。

## 样本叙事弧（高密度抽帧提炼 · 可迁移结构）

对样本约每 0.5s 抽帧（fps=2）后，口播/字幕推进大致为：

1. **赌注钩子**：如果你想多活几年……  
2. **主体登场**：机制对象用「我」说话（不是老师在讲课）  
3. **终身劳动**：我出生就开始工作了  
4. **尺度具象**：成年后只有拳头大（日常物对照）  
5. **委屈/处境**：一生窝在狭小胸腔  
6. **节律自述**：我放慢速度 / 其实我挺厉害的  
7. **生活背叛蒙太奇**：烟、吃爽了、一生气一焦虑……（B-roll 对照负荷）  
8. **社会隐喻一锤**：这就是强制加班（科学机制 × 职场话）  
9. **预警/反讽收束**：等你哪天真的感觉到；你开心就好；自然不需要你去做什么  
10. **感官 CTA**：听听我的跳动 → 评论区自然涌出「谢谢你 / 对不起」

**可迁移公式**：`赌注钩 → 主体拟人 → 尺度/处境 → 生活对照 × N → 一个社会/古典隐喻 → 关系型 CTA`  
禁止写成「定义→结构→功能→注意事项」课件提纲。

## 如何跨界（举例菜单 · 每条至少选 1–2 种）

跨界 = **机制真相** 与 **另一语义场** 对照，不是两段拼盘。选题时从下表挑，并写清「对照点」：

| 跨界类型 | 怎么接 | 合格示例（勿照抄） |
|----------|--------|-------------------|
| A. 拟人/关系 | 对象用「我」说话；人或观众对它说谢谢/对不起 | 心脏自述值夜班；人对胰岛说「今天害你加班了」 |
| B. 职场/社会 | 机制 ≈ 排班、强制加班、后台服务、客服值守 | 心跳不停 = 强制加班；肝脏解毒 = 通宵清缓存 |
| C. 饮食作息 | 一顿饭/一根烟/熬夜对应负荷变化 | 「你是吃爽了」对上消化与循环代价 |
| D. 情绪压力 | 生气焦虑瞬间推高负荷 | 怒吼画面切回发光脏器加速 |
| E. 尺度具象 | 拳头、一粒米、一滴、一拍 | 心脏≈拳头；胰岛β细胞≈微型工厂夜班岗 |
| F. 古典/历史/艺术 | 一句典籍、一件器物、一段乐理对照机制 | 《内经》时序 × 肾的夜班；爵士空一拍 × 换气 |
| G. 运动/旅行场域 | 发球、登顶、深潜、长途与瞬时负荷 | 网球发球瞬间心率 ×「往高处打」的人生节奏 |
| H. 器物/工程隐喻 | 泵、滤芯、电网、冷却系统 | 肾脏≈滤芯；线粒体≈充电宝阵列 |

**同批轮换**：六条选题至少覆盖 2 种跨界类型 + 2 个不同机制主体；勿全是「心脏×谢谢你」复刻。

## 硬约束
1. **叙事优先于名词堆砌**：先情绪与对照，再落 1–2 个可感知机制点（节律、负荷、恢复）；禁止课堂念稿。  
2. **真人个案硬门槛（与 `vivid-anti-boring` 同开）**：跨界科普**禁止**只剩拟人/隐喻/三点清单。每条必须插入至少 1 个**生动个人/身边案例**（猝不及防、年轻中招如渐冻症等可含但不限、饭桌劝不动父母的具体对话……）。知识大家听过；**案例不是人人知道**——用案例把机制钉进记忆。  
3. **跨界必须服务机制**：隐喻句删掉后若机制仍不清楚，说明还没讲清，不是跨界失败就堆典。  
4. **审核友好**：学者向观察；故事唤醒 ≠ 恐吓营销；**不**开处方、**不**承诺疗效、**不**对观众下诊断；变现走深度交流/体系课，不走问诊。  
5. **视觉/分镜**：可电影感透视、半透明腔体、发光机制体；穿插生活 B-roll **与个案相关的具体物件/场景**；口播字幕短句有情绪；禁挂图墙与诊疗恐吓画面。  
6. 与 `review-safe-voice`、`cultural-diversity` 同时生效时：取材开放（含宋朝与影视），同批勿六条复读同一主角/同一案例类型即可。  
7. 与 `contrast-reversal-climax` 叠用时：允许「感谢→愧疚→和解」或「炫耀健康→猝不及防」式情绪反转，高潮不限定仰慕。

## 合格方向（主体可换 · 每条仍须带个案）
- 胰岛「值夜班工厂」× 节气饮食节律 × **一个「还在说自己血糖没事」的饭局对照**（跨界 B+C+F，不谈降糖药方）  
- 肾「深夜仍在排班」× 出差时差 × **同事红眼航班后倒下的切片**（B+C）  
- 肠道菌群「城里小商贩」× 外卖周 × **连续七天只点麻辣烫的人设故事**（A+B+C）  
- 心肺负荷 × 网球发球 × **球场上还在吹牛体力好的人**（E+G）  
- 「谢谢你，心脏」拟人弧 × **必须再接一个真人故事**，禁止只剩器官独白

## 自检
- [ ] 是否有情绪共鸣，而不只是名词解释？  
- [ ] **有没有具体个案**？删掉隐喻后故事还在不在？  
- [ ] 跨界类型能否点名（上表字母），且对照服务机制？  
- [ ] 主体换成另一机制对象，结构是否仍成立？  
- [ ] 是否审核友好、无诊疗承诺？  
- [ ] 结尾是否留给「谢谢你 / 对不起 / 听听……」一类关系型互动，而非纯知识点复述？
