# 反差·反转·高潮弧 Skill

## 用途
短视频 / 图文选题与文案默认可启用。目标不是堆百科，而是做出**有趣、新奇、有反差**的一条内容：**身份有反差，情绪有反转**；反转与情绪变化可发生**一次或多次**，次数与落点跟文案需要走，不要套死单一结局。

参考原型（可换人设，勿照搬）：低地位角色（如服务员）× 高认知身份（如某领域博士）→ 现场用可核对的专有观察讲通日常场景 → 对面人情绪/态度发生可见变化。原型里可以是轻视→佩服，但**不强制「仰慕」**，也可是嘲讽→认真、冷漠→好奇、自信→动摇、愤怒→释然等。

## 硬约束（两根柱子）

1. **身份反差**：标题或开场必须有身份 / 预期错位（A 不该出现在 B；或标签与行为打架）。
2. **情绪反转**：正文或分镜里至少有一处**可见的情绪/态度翻转**；允许中途再翻一次或两次（小反转叠大反转），**以文案节奏为准，不规定必须翻几次、必须以仰慕收束**。

## 推荐结构（可伸缩，不是死三拍）

### A）开场反差钩子（停滑）
- 第一句抛身份错位或偏见（看不起、不信、自我调侃、围观起哄都行）。
- 忌正确但无聊的自我介绍；先让观众感到「不对劲」。

### B）中段：专有词升压 + 可插小反转
- 把日常动作翻译成**可核对的专有表述**：
  - 优先：**具体指标 + 方向 + 幅度**（例：「大脑葡萄糖消耗比常人高出 20%」「生理坚持极限不超过 135 秒」）。
  - 少用含糊笼统词（不要只说「血糖不对」「很累」就收工）。
- 专有词服务叙事与反差，不是科普课；一镜/一段 **1 个主术语 + 1 个可感知场景结果**。
- 此处可插入**小情绪反转**（例：刚被说服 → 又起疑 → 再被下一证据打脸），看篇幅决定。

### C）高潮：情绪落点（形态自选）
- 高潮落在**人物情绪/关系变化**或**认知翻转**，不是再抛更难的术语墙。
- 落点示例（择一或组合，勿写死）：佩服、认真听完、想继续问、自嘲打脸、沉默、把话转给需要的人、决定试半成品动作……**「仰慕」只是其中一种，不是默认必写**。
- 若文案需要双高潮：可先小反转再大反转；若篇幅短：一次干净反转即可。
- 收束可接半成品行动 / CTA（仍遵守 `hook-solution-cta`）；不要突然变说教课。

## 与半成品解法的关系
- 本 Skill 管**反差修辞与情绪戏剧弧**；`hook-solution-cta` 管咨询闭环。
- 中段专有词 = 半成品解法的「证据感」；完整方案仍留白。

## 强监管赛道注意
- 健康 / 医学向：学者观察口吻；机制与指标**不承诺疗效、不开处方、不点名诊断**。
- 可用「葡萄糖消耗 / 皮质醇水平 / 生理时限」等机制词；避免「治好 / 必降糖 / 替代就医」。
