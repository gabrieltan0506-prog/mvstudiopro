# 反差·反转·高潮弧 Skill

## 用途
短视频 / 图文选题与文案默认可启用。目标不是堆百科，而是做出**有趣、新奇、有反差**的一条内容：观众先带偏见，再被具体能力说服，最后态度翻转。

参考原型（可换人设，勿照搬）：低地位角色（如服务员）× 高认知身份（如某领域博士）→ 用现场可验证的专有观察讲通一个日常场景 → 对面人从轻视到仰慕。

## 高度需求结构（三拍）

### 1）开场反差钩子（停滑）
- 标题 / 第一句必须是**身份或预期错位**：A 身份不该出现在 B 场景（例：「为什么博士适合做服务员」）。
- 第一屏立刻抛出偏见或质疑（对面人看不起、围观不信、自我调侃都行）。
- 忌正确但无聊的自我介绍；先让观众站在「看不起」这一边。

### 2）中段：专有词升压（新奇，不是空唬）
- 把日常动作（点菜、等餐、加菜、服务节奏）翻译成**可核对的专有表述**：
  - 优先：**具体指标 + 方向 + 幅度**（例：「大脑葡萄糖消耗比常人高出 20%」「生理坚持极限不超过 135 秒」）。
  - 少用含糊口语笼统词（例：不要只说「血糖不对」「很累」就收工）。
- 专有词要服务叙事：解释「他为什么在这个岗位更强」，不是科普讲座。
- 一镜/一段 **1 个主术语 + 1 个可感知场景结果**（如加菜、停加菜、防御解除），忌一分钟塞十个名词。
- 可混用生理 / 心理 / 消费行为术语，但必须落回眼前这一桌、这一个人。

### 3）高潮：态度反转（轻视 → 仰慕）
- 必须有**对面人的可见态度转折**：质疑 →「你也太厉害了」→「我现在有点仰慕你了」（措辞可换，弧线不可缺）。
- 高潮落在**人物关系**，不是再抛一个更难的术语。
- 收束可留一句反差金句或半成品行动（仍遵守钩子·解法·CTA Skill）；不要突然变成说教课。

## 与半成品解法的关系
- 本 Skill 管**戏剧弧线与反差修辞**；`hook-solution-cta` 管咨询闭环。
- 中段专有词 = 半成品解法的「证据感」；完整方案仍留白给私信/咨询。

## 强监管赛道注意
- 健康 / 医学向：用学者观察口吻；写机制与指标时**不承诺疗效、不开处方、不点名诊断**。
- 可用「葡萄糖消耗 / 皮质醇水平 / 生理时限」等机制词；避免「治好 / 必降糖 / 替代就医」。
