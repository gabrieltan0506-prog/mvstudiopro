# 文化素材开放池 Skill

## 用途
凡人设含传统文化、古典文学、历史、中西对照、生命美学、影视跨界时启用。  
**不是**禁止史记/爵士/银发/边塞/影视，也**不是**避开宋朝——而是打开素材池。  
**更重要**：人设里的「唐诗 / 爵士 / 典籍」是**可用容器**，不是每条选题的**必交作业**。

## 硬约束
1. **开放取材**：周秦汉魏晋唐宋元明清至近现代、当代皆可；典籍、历史人物、平民与贵族、器物事件、**影视剧/电影桥段**、**当下生活物件**均可。
2. **IP DNA ≠ 每条复读**：人设写了「缝合唐诗、爵士与艺术」时，同批**最多 2 条**以典籍/爵士/博物馆当主容器；其余须换热播剧、关系对话、通勤/冰箱/宵夜、职场局、亲子饭桌、畅销品等**生活化切口**。
3. **禁「读论文公式」选题**（b2.pdf 反面）：`读《××》领悟××` / `哈佛实验室揭秘××` / `用××观观察××` / 标题堆 `皮质醇·线粒体·HRV·内皮` 却无可拍生活场面。这类**必须重写**。
4. **优先热播与生活**：《藏海传》《庆余年》《繁花》式高压剧、路边摊、加班回消息、父母不听劝——比「再读一遍苏轼/史记」更能停滑。典籍可以写，但要写成**人在干嘛**，不是文献综述。
5. **宋朝可写**：范仲淹、苏轼等皆可——**同一批六条不要多数条都钉死同一人**即可。
6. 文化是容器：每条仍须客户痛点 → 钩子 → 半成品解法 → CTA。

## 素材池（交叉抽取；热播/生活优先于炫学）
- **当下媒介（优先）**：热播剧/电影/综艺桥段、短视频热梗改写、畅销消费品
- **日常与关系**：通勤、冰箱、宵夜摊、亲子餐桌、银发陪伴、职场高压局
- **典籍/人物（配额内）**：史记、内经、唐宋人物等——服务痛点，勿炫学
- **艺术听觉（配额内）**：爵士现场、黑胶、展陈——勿六条黑胶书房

## 合格例 vs 不合格例
| 合格（想看） | 不合格（读论文） |
|-------------|-----------------|
| 别被《藏海传》权谋骗了：高压局里先保命的 3 个身体动作 | 读《史记》治不好你的病，但能救你的命：英雄史观抗压实验 |
| 凌晨路边摊：先别劝戒宵夜，先看你身体在跟你吵什么 | 揭秘哈佛实验室针对慢病人群的 5 招血管减龄法 |
| 父母不听劝？别说「注意身体」，换一套战国说客也不用的说法 | 范仲淹也焦虑？《岳阳楼记》里的顶级心脏养护方案 |

## 自检
- [ ] 同批是否 ≤2 条「典籍/爵士主容器」？  
- [ ] 是否至少 2 条热播或纯生活切口？  
- [ ] 有没有「读《》领悟 / 哈佛实验室揭秘」式标题？有则重写  
- [ ] 是否服务人设痛点，而非炫学？
