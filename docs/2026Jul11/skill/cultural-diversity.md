# 文化素材开放池 Skill

## 用途
凡人设含传统文化、古典文学、历史、中西对照、生命美学、影视跨界时启用。  
**不是**禁止史记/爵士/银发/边塞/影视，也**不是**避开宋朝——而是打开素材池，按人设自由取材，同批不要六条复读同一主角。

## 硬约束
1. **开放取材**：周秦汉魏晋唐宋元明清至近现代、当代皆可；典籍、历史人物、平民与贵族、器物事件、**影视剧/电影桥段**均可。
2. **宋朝可写**：范仲淹、文天祥、王安石、司马光等皆可；苏轼/李清照也可写——**同一批六条不要多数条都钉死同一人**即可。
3. **影视健康向示例（勿照抄）**：《藏海传》里藏海如何在高压权谋中维持身体精力旺盛——**只给 3 个要点**（数目要克制，见 `hook-solution-cta`）。
4. **反差选题欢迎**：如「博士生去应徵服务员」式身份错位标题；也可由模型按人设**新创桥段**，不限时代。
5. 文化是容器：每条仍须客户痛点 → 钩子 → 半成品解法 → CTA。

## 素材池（交叉抽取，非禁令名单）
- **典籍**：史记、战国策、诗经楚辞、论孟庄、内经千金本草、通鉴、唐诗宋词元曲、明清小说笔记、当代散文……
- **人物**：帝王将相、文人士大夫、名医工匠、商贾游侠、市井平民、女性作者、边疆使者、近现代与当代公众人物（审核友好）
- **影视/跨媒介**：热播剧、电影、纪录片中的情绪/健康/压力桥段，改写为本人设可讲的切口
- **生活对照**：运动、旅行、职场、家庭沟通、音乐现场等——按人设有则用，无则勿硬塞

## 自检
- [ ] 是否服务人设与痛点，而非炫学？  
- [ ] 同批是否避免六条同一主角复读？  
- [ ] 若用影视，是否改写成可拍、可咨询的半成品，而非剧情剧透墙？
