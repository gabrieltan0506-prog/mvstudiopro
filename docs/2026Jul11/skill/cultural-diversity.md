# 文化素材开放池 Skill

## 用途
凡人设含传统文化、古典文学、历史、中西对照、生命美学、影视跨界时启用。  
**不是**禁止史记/爵士/银发/边塞/影视，也**不是**避开宋朝——而是打开素材池并**强制轮换**。  
人设里的「唐诗 / 爵士 / 典籍」是**可用容器**，不是每条必交作业。

## 为什么老是苏轼？（根因与对策）
模型爱拿「苏轼 × 哈佛 × 血管」当安全跨界模板。  
**对策**：同人限条 + 轮换池 + 已出现勿再写 + 放松向优先。苏轼**可以写**，但不能当默认男主。

## 硬约束（同人 / 轮换）
1. **同人限条**：同一历史人物或同一影视角色，同批默认**只准 1 条**。仅当角度**截然不同**且场景标题完全不撞，才允许第 2 条；禁止同人同观点换皮。  
2. **禁止默认苏轼**：不得条条苏东坡。须从下面轮换池抽人，**先换人再换作品**（一首作品就能长出很多切口）。  
3. **已写过的人**：本批已出现的人物，其余条目**不要再出现**；跨次若上下文高频同一人，除非**时事热点 / 当季节令 / 用户点名**，否则换人。  
4. **IP DNA ≠ 每条复读**：同批**最多 2 条**以典籍/爵士/博物馆当主容器；其余须换热播、关系、烟火、畅销品。  
5. **禁读论文公式**：`读《××》领悟…` / `哈佛实验室揭秘…` / 标题堆机制名词却无生活场面 → 必须重写。  
6. **放松向**：观众忙完一天是来放松的——先有戏、有局、有情绪；不是来听课。

## 人物轮换池（交叉抽取，勿钉死第一位）
- **三苏**：苏洵、苏轼、苏辙（轮流，禁止永远苏轼）  
- **唐宋八大家**：韩愈、柳宗元、欧阳修、苏洵、苏轼、苏辙、王安石、曾巩  
- **其他常客可轮**：李白、杜甫、白居易、李清照、辛弃疾、范仲淹、司马光、陆游…  
- **史传人物**：司马迁、刺客游侠、医者工匠、市井平民  
- **影视角色**：热播剧高压局（如藏海）——与历史人物分开计数

## 素材优先级
1. 当下生活与关系  
2. 热播/影视桥段  
3. 烟火/旅行/运动场域  
4. 典籍人物（配额内 + 轮换）

## 合格 vs 不合格
| 合格 | 不合格 |
|------|--------|
| 苏辙在兄弟阴影里怎么活：高压家里排行老二的身体账单 | 又一篇苏轼冷灶 × 血管弹性 |
| 别被《藏海传》骗了：高压局保命的 3 个动作 | 读《史记》英雄史观抗压实验 |
| 韩愈谏佛骨被贬：一张嘴换一条命，现代人哪门课敢这么讲 | 哈佛博士再读一遍东坡定风波 |

## 自检
- [ ] 同人是否默认 ≤1？  
- [ ] 是否又默认苏轼了？有则换三苏其他或八大家  
- [ ] 本批已出现的人是否还在后面条目复读？  
- [ ] 是否至少一半像放松向短视频，而不是课堂？  
- [ ] 有没有「读《》领悟 / 实验室揭秘」论文题？
