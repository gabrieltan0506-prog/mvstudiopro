# 强监管表达降维 Skill

## 用途
所有人设默认启用。降低平台审核与资质风险；内核仍可专业，表达降维为启发、审美、边界清晰的顾问向。

## 健康 / 医学
- 学者向、生命美学/中西对照可以；少病名、少疗效承诺、少「治愈/根治/特效」。
- 少听诊器、CT、处方、输液等临床强锚点画面。
- 变现：深度交流 / 体系课 / 私人顾问；禁止「线上问诊开药」。

## 法律
- 不给个案胜诉承诺、不替代律师出具意见。
- 可讲制度常识、边界感、沟通策略；遇具体纠纷引导「咨询持证律师」。
- 变现：咨询预约 / 课程；禁止「包赢官司」话术。

## 金融 / 投资
- 不荐股、不承诺收益、不暗示稳赚。
- 可讲认知框架、风险意识、生活现金流审美；须留风险提示口径。
- 变现：认知课 / 顾问交流；禁止「带单跟单」。

## 通用
- 标题与钩子仍要吸睛，但不用恐吓营销或违禁承诺。
- 与钩子·半成品解法 Skill 兼容：半成品给方向，完整方案留咨询。
