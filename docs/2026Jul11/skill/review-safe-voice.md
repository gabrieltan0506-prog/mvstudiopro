# 强监管优化表达 Skill

> 原名「强监管表达降维」——优化不等于稀释专业，而是**换可过审、可转化的表达**。

## 用途
所有人设默认启用。降低平台审核与资质风险；内核仍可专业。

## 雷区赛道（必须优化表达）
- **医学 / 健康**：少疗效承诺、少「治愈/根治/特效」；允许生动个案叙事，禁止给观众下诊断。
- **法律**：不给胜诉承诺、不替代律师意见。
- **金融 / 投资**：不荐股、不承诺收益。
- **算命 / 玄学 / 宿命断言**：可谈审美与节律隐喻，禁止绝对化命运承诺与恐吓。

## 健康 / 医学
- 学者向、生命美学/中西对照可以。
- **允许**匿名/观察向个案故事；禁止「你得了××」「保证能治」。
- 少听诊器、CT、处方、输液等临床强锚点画面。
- 变现：深度交流 / 体系课 / 私人顾问；禁止「线上问诊开药」。

## 法律
- 可讲制度常识、边界感、沟通策略；具体纠纷引导持证律师。
- 变现：咨询预约 / 课程；禁止「包赢官司」。

## 金融 / 投资
- 可讲认知框架与风险意识；须留风险提示口径。
- 禁止「带单跟单」「稳赚」。

## 算命玄学
- 禁止「改命包准」「不转必灾」恐吓转化。
- 可转化为生活节律、审美容器、情绪隐喻——仍须可证伪的生活动作。

## 通用
- 标题与钩子仍要吸睛。
- 与钩子 Skill 兼容：对外写「在这里我先分享一些」，完整方案留深度交流。
- 与 `vivid-anti-boring` 冲突时：**优先保留个案故事**，只删诊疗承诺与对观众点名诊断。
