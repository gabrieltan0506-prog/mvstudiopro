# 畅销品科普变现闭环 Skill

## 用途
与 `4season-fmcg-popsci` + `label-debunk-copy` + `authority-cite-endorsement` 同开时启用。  
解决：**视频好看但不变现**——把「会看包装的人」收成「愿意为生命管理付钱的线索」。

本 Skill **不重写**打脸句式与运镜；只规定**商业漏斗、CTA、私域话术与禁区**。  
开场火花与半成品数目仍遵守 `hook-solution-cta`；审核边界遵守 `review-safe-voice`。

## 一句话闭环
**打脸畅销品 → 证明会翻译证据 → 只给半成品 → 完整路径进私域（顾问 / 体系课 / 沙龙）**

钱在**高客单解释权**，不在雪糕/零食佣金。

## 四层漏斗（写进文案结构）

| 层 | 观众得到 | 内容动作 | 商业动作 |
|----|----------|----------|----------|
| 1 停滑 | 原来包装在骗感觉 | 四季弧 + 打脸句 | 播放/关注 |
| 2 信任 | 这人懂指南/国标，不装神医 | 权威一句 | 信任资产 |
| 3 半成品 | 会翻表、会算一天糖盐、会对比两款 | 2–3 点，故意留白 | 评论关键词领清单 |
| 4 成交 | 家庭膳食/礼盒/儿童零食边界/高压节律 | CTA 私信深聊 | 顾问·课·沙龙 |

**硬纪律**：视频里**禁止**把可替代咨询的完整方案讲完；讲完就没有第 4 层。

## 产品阶梯（按人设选用，勿一条视频卖三样）

### L0 钩子产品（几乎必做）
免费或极低摩擦，只换线索：
- 家庭包装识读清单 / 节令礼盒避坑卡 / 添加糖一日对照卡  
- CTA：`评论区打「识读」/「韧性」/「礼盒」` → 私发 PDF 或长图  

### L1 主变现（默认主推）
- 私人生活方式顾问（**非问诊开药**）  
- 家庭膳食与节律方案（按周/季）  
- 「高压人群」恢复系统咨询（作息·饮食·边界，不写处方）  

### L2 场域变现
- 线下生命美学 / 康养哲学沙龙（门票 → 会员）  
- 企业高管小班（效能与恢复，审核友好表述）  

### L3 不要当主变现
- 雪糕、零食、饮料 **佣金带货**（与「不划算」叙事打架，人设掉价）  
- 若碰商品：只做**选品逻辑课**或高客单定制礼盒顾问，不当直播间卖货主  

## 视频结尾 CTA 模板（选 1）

1. **清单线索**：`想要《家庭包装识读对照表》的，评论区打「识读」，我私你；有家里具体场景的，我们深聊。`  
2. **顾问预约**：`半成品你自己就能做；若要按你家冰箱和作息定制路径，私信「顾问」预约。`  
3. **沙龙/课**：`下期沙龙名额，评论「沙龙」——先聊清楚你是谁、怕什么，再决定来不来。`

禁忌：`点击小黄车`、`保治`、`线上开药`、`转发送课包赢健康`。

## 私域三问（线索进来后 24h 内）
1. **谁在买？**（自己 / 孩子 / 父母 / 公司礼盒）  
2. **最怕什么？**（胖、蛀牙、三高焦虑、被营销骗、年节丢脸）  
3. **愿意改到哪一步？**（只换品牌 / 愿意算一天糖盐 / 要系统性方案）  

答完再推 L1/L2；**禁止**一进私域就报价甩课表。

## 平台分工（同批可写进 publishingAdvice）
- **抖音**：打脸钩子 → 评论关键词线索  
- **小红书**：图文对照表沉淀搜索 → 咨询  
- **视频号**：转发感 → 私域成交  
- **B 站**：稍深标签/国标逻辑养信任 → 再导顾问  

## 与三件套如何叠用
1. `4season-fmcg-popsci`：挂品、弧、镜头  
2. `label-debunk-copy`：怎么说；表不清走盲区  
3. `authority-cite-endorsement`：收束前 1 句权威  
4. **本 Skill**：结尾 CTA 类型 + 线索词 + 禁止讲完  
5. `hook-solution-cta`：痛点/爽点前置与半成品数目  

## 文案自检（变现专用）
- [ ] 是否故意留了咨询才补得齐的缺口？  
- [ ] CTA 是否指向清单/顾问/沙龙，而非小黄车？  
- [ ] 有无「疗效/问诊/开药」违禁变现话术？  
- [ ] 私域路径是否写得出「三问」而不是直接逼单？  
- [ ] 同批是否避免六条全是食品糖盐、毫无 L1 场景多样性？

## 合格结尾示例（可改写）
> 自己能做的到这里：翻面、看每 100 克、对照指南那一句。  
> 你家孩子零食柜、年节礼盒、或高管作息怎么收束，评论「识读」——表我发你；要定制路径，我们私聊。
