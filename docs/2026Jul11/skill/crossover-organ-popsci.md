# 跨界器官科普叙事 Skill

## 用途
人设含生命科学 / 慢病观察 / 中西医学原理 / 健康审美时启用。把「干巴巴解剖课」改成**有情绪、有画面、愿转发**的跨界科普。参考样本气质（勿照搬心脏单题）：器官拟人第一人称或「谢谢你/对不起」式共鸣 + 电影级解剖视觉，再挂古典/历史隐喻。

## 硬约束
1. **器官可轮换**（举例，非穷尽）：心脏、肺、胰腺（胰岛）、肾、肝、肠、血管……同一批选题**不要**六条都写心脏。
2. **叙事优先于名词堆砌**：可用器官「自述」或人对器官说话（感谢/愧疚/和解），再落到 1–2 个可感知机制点（节律、负荷、恢复），**禁止**课堂念稿式「首先结构其次功能」。
3. **跨界挂钩**：每条至少一处古典 / 历史 / 艺术 / 生活场域隐喻（典籍原句、器物、乐理、旅行、运动皆可），与器官机制**对照**，不是两段拼盘。
4. **审核友好**：学者向观察；**不**开处方、**不**承诺疗效、**少**点名病种恐吓；变现走深度交流/体系课，不走问诊。
5. **视觉/分镜**：可写电影感解剖透视、半透明胸腔、发光脏器等「科普大片」质感；口播字幕宜短句、有情绪，勿长段定义。
6. 与 `review-safe-voice`、`cultural-diversity` 同时生效时：器官题仍须拉开朝代/典籍，勿默认苏轼/李清照扎堆。

## 合格方向示例（勿照抄）
- 胰岛「值夜班的工厂」× 节气饮食节律（不谈降糖药方）  
- 肾「深夜仍在排班」× 出差时差与《内经》时序  
- 肝「默默解毒的后台」× 高压周会后的假休息  
- 肺「每一次换气」× 爵士空一拍 / 庄子虚室（机制点到为止）

## 自检
- [ ] 是否有情绪共鸣，而不只是名词解释？  
- [ ] 器官是否可换成另一脏器仍成立（证明结构可迁移）？  
- [ ] 古典/历史是否服务机制，而非炫学？  
- [ ] 是否审核友好、无诊疗承诺？
