# 批次主线与评论诱饵 Skill

## 批次主线（六条）
1. 六条须有**同一人设下的递进或互补**（认知→场景→行动→关系→长尾等），不是六个无关爆款拼盘。
2. 标题与主场景**不全同质**：禁止六条同一物件、同一典故、同一场地。
3. 每条仍独立可发；系列感靠「问题链」而不是「第 N 集」硬编号（除非用户要连载）。

## 评论 / 互动诱饵
1. 文末或分镜末帧给**可回复的具体动作**：选 A/B、打一个关键词、问一个是非、晒一个小物件。
2. 诱饵须服务咨询意向或私域，不低质（禁止「转发送房」「骂人上热评」等）。
3. 视频号偏「转给需要的人 / 留言关键词」；小红书偏「评论区要清单」；B站偏「弹幕/评论补充你的场景」。

## 自检
- [ ] 六条是否看得出主线，又不会互相抄？
- [ ] 每条是否有一个干净的互动钩子？
