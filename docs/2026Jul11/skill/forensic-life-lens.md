# 法医视角·还怎么活 Skill

## 用途
当选题需要**稀缺医学视角**、又要停滑与共鸣时启用。  
核心颠覆：**法医不是猎奇解剖号**，而是用「死亡侧看见的规律」翻译「还活着的人怎么少留遗憾」。

适合人设：医学/生命科学/生命美学/高压人群顾问（如哈佛医学背景）。  
**不要**去硬刚 @法医刘良 类「解剖间日常」头部号；做**视角容器**，不是第二法医号。

## 一句话公式
`日常赌注钩 → 法医式一句真相（不血腥） → 生活对照 → 2–3 个半成品保命/恢复动作 → 权威一句 → 评论关键词领清单`

观众刷完应更想**好好活**，不是更想看尸检。

## 硬禁（审核 + 人设）
1. 禁止：尸检/解剖特写、血腥、真实命案复盘、可识别当事人细节。  
2. 禁止：对观众下诊断、恐吓「你明天必死」、线上开药/问诊。  
3. 禁止：标题党「尸体里挖出××」猎奇；可用悬念，但落点必须是**还怎么活**。  
4. 允许：匿名观察向遗憾叙述、热播探案剧一句钩子、生活风险场景。  
5. 简体中文；成稿禁导演名/片名致敬。

## 叙事弧（可迁移）
1. **日常局**：安全带、酒桌、熬夜回消息、广场舞口角、孩子发烧、健身房逞强…  
2. **法医式一句**：报告/档案里常看见的规律（原则级，不写具体案号）。  
3. **对照**：观众以为是「倒霉/玄学/矫情」，其实是负荷与窗口期。  
4. **半成品 2–3 点**：立刻能做的边界动作（不是完整急救课）。  
5. **权威一句**：膳食/活动/睡眠/急症求助公众口径（见权威 Skill）。  
6. **CTA**：评论「保命」「清单」领家庭急症决策卡 → 私域顾问（非案情付费课）。

## 选题菜单（每批挑 1–2 条主用，勿十条堆一条视频）

### 1）安全带不是怂
钩：高速事故后还能坐着说话的人，法医叙述里最常强调哪句习惯？  
半成品：上车三秒系好；后排也系；短途不豁免。

### 2）口角几分钟
钩：情绪顶穿的往往不是「被打死」，是剧烈波动后的心血管事件窗口。  
半成品：冲突先散场；老年人争执降级；胸痛不硬撑。

### 3）「还能自己打 120」的两小时
钩：以为是小病拖着，呼吸通道类风险最容易被外行耽误。  
半成品：呼吸费力/声音变了走急诊；不自行加码拖；家人分工谁打车谁打电话。

### 4）权谋剧「突然暴毙」
钩：《藏海传》式一怒而终 → 现代对照是急性事件，不是玄学。  
半成品：怒前离开现场；冷水面部刺激；事后不当「没事」。

### 5）凌晨回消息的身体账单
钩：熬夜后突发在档案叙述里并不罕见；你以为拼事业，身体只记负荷。  
半成品：固定关机点；咖啡截止；加班周强制恢复窗。

### 6）酒桌劝酒的体面代价
钩：年夜饭最危险的常常不是菜，是「再满上」。  
半成品：拒酒三句；醉酒侧卧；不让醉的人独自回家。

### 7）头盔松着≈没戴
钩：电动车/雨夜「差那一下」会出现在太多可避免的遗憾里。  
半成品：贴合自检；雨夜降速；反光条。

### 8）孩子发烧你还在开会
钩：「拖」本身会出现在很多可避免叙事里——不是恐吓，是决策窗口。  
半成品：何时急诊的家庭决策树；会中请假话术；药箱常备边界（不讲处方疗效）。

### 9）健身房逞强一组
钩：年轻人猝倒不全是「遗传」；脱水+刺激物+硬扛是常见组合叙述。  
半成品：新动作减半重量；胸闷即停；补水节奏。

### 10）「没事就是焦虑」的漏诊窗口
钩：情绪问题与急症可以并行，不互斥；胸痛/一侧无力/言语不清先当急症。  
半成品：红旗症状清单；先急诊再谈情绪管理；伴侣分工。

## 运镜与画面（轻）
- 开场：生活场景特写（安全带咔哒、酒杯、手机凌晨、头盔扣具）——**不要**解剖台。  
- 中段：信息卡「窗口期 / 负荷 / 三步动作」；人物表情认真或心疼，勿恐怖片光。  
- 收束：家庭清单封面或评论关键词大字。  
- 与 `director-craft` 同开时，本题材优先本 Skill 画面禁令。

## 变现（叠 `fmcg-popsci-monetize` / `hook-solution-cta`）
- L0：家庭急症决策卡 / 保命习惯清单（评论词领取）  
- L1：高压人群恢复与家庭风险边界顾问（非问诊）  
- **禁止**：案情付费课、血腥资料包、小黄车猎奇周边  

私域三问：家里谁最爱硬扛？最近一次差点忽略的信号？愿不愿意做恢复日程？
