# 四季畅销品轻科普 Skill（弧 + 运镜合一）

## 用途
选题挂在**当季可感知的畅销消费品**上做轻科普时启用（食品优先，也可延到季节性日化/礼盒，但仍须可翻包装或可对比）。  
本 Skill = **叙事弧 + 画面语法**（原 `summer-fmcg-popsci` + `food-popsci-lens` 合并）。  
文案句式与「标签看不出时怎么办」见 `label-debunk-copy`；权威一句背书见 `authority-cite-endorsement`；**变现闭环**见 `fmcg-popsci-monetize`。

目标：停滑 + 认知打脸 + 立刻能用（翻包装 / 看营养成分 / 对标准代号），不是课堂念稿，也不是器官拟人（那是 `crossover-popsci`）。

参考样本方法：`~/Downloads/2026Jul12/ice.mp4`（老洋人科普 · 高密度抽帧）；**公式四季通用，品类随季节换**。

## 四季畅销品菜单（每条选题只挂 1 个主品，可很多写）

### 春
花茶/新茶礼盒、清火饮料、酸奶杯、轻食代餐、花粉季相关洗护（若做人设允许）、清明踏青零食。

### 夏（样本最熟）
雪糕/冰棍/冰淇淋、冰镇饮料、果啤与无醇、防晒霜与晒后（非食品时改「看成分表/防晒指数」）、解暑粥/凉粉类现制对照。

### 秋
月饼/粽子类节令糕点、坚果礼盒、新米与杂粮、暖饮、润燥向零食（仍要打脸营销话术）。

### 冬
火锅底料/复合调味、腊味礼盒、热可可与奶茶粉、糖果巧克力、年货坚果炒货、保温杯场景下的高糖热饮。

### 全年常青（不受季节锁死）
儿童零食、代糖饮料、「0糖/低脂/植物基」宣称款、方便速食、蛋白棒/代餐奶昔、宠物粮（若人设相关）、厨房调味酱。

**可迁移公式不变**：换品名、换体感钉子、换季节场景即可连写多条。

## 样本叙事弧（可迁移）

1. **日常赌注钩**：如果每天都来一根/一包/一杯 ____…  
2. **反驳预埋**：先别急着反驳  
3. **翻包装动作**：嘿，你看过配料表 / 营养成分表吗？  
4. **排序法则**：配料按含量从高到低（能翻到才用）  
5. **认知打脸**：以为 X → 其实 Y（或走「标签盲区」支线，见 `label-debunk-copy`）  
6. **工业/工艺底色**：2–3 个角色点到为止（乳化/增稠/香精/植脂/复合调味等）  
7. **调包或宣称拆穿**：看起来高级的那一层（脆皮、夹心、「奶味」、节日礼盒）  
8. **量感钉子**：具体克数 / 占一日添加糖上限的几分之几  
9. **体感或生活反常识**：夏天可「越吃越热」；冬天可「越喝越渴/更燥」；全年可「以为在补，其实在灌糖」  
10. **儿童/银发线（可选）**：口味养成、牙齿、给长辈买礼盒的心疼——不做诊疗恐吓  
11. **权威一句**：从 `authority-cite-endorsement` 食品栏挑 1 条，勿堆  
12. **总账收束**：不划算；钱进了包装与广告  

## 运镜语法（食品/货架主用；日化改「成分表特写」）

### 开场停滑（0–3s）
- 断裂/拉丝/撕袋/揭盖/倒出液体特写；或一口咬入。  
- 叠字：日常赌注或反驳预埋。

### 翻标签段（有表才拍）
- 手持背面：配料表或营养成分表占满画面；圈前 2–3 名或能量/糖/钠。  
- 大字：「从高到低」或「先看每 100 克」。

### 标签不够用时（盲区支线画面）
- **营养成分表特写**（糖/钠/能量）+ 对照「一日上限」手卡。  
- **产品标准代号 / 类别名**特写（如冰淇淋分类、植脂 vs 全乳脂话术）。  
- **宣称对比**：正面「0蔗糖」大字 ↔ 背面碳水/甜味剂小字分屏。  
- **抽检/通告类**只做信息卡风格，勿伪造红头文件画面。

### 工业与量感
- 搅拌罐、淋面、倒糖浆/盐勺入碗——把「十几克」拍成看得见。  

### 体感与关系
- 出汗/嗓子干/孩子要买/老人拆礼盒——轻，勿恐怖片。  

### 收束
- 货架矩阵或同款回环；大字「怎么算都不划算」。

### 灯光
- 食品：食欲暖光；工业：冷金属；打脸字：高对比短停。成稿禁导演名/片名。

## 硬约束
1. **品类必须当季或常青可感知**；禁止空讲「健康生活」不挂具体 SKU/品类。  
2. **禁止深医学词墙**；用体感、包装、数字说话。  
3. **必须有一个立刻能做的动作**：翻配料 / 看营养成分 / 对比两款前三名 / 核对宣称与标准名。  
4. **标签看不清或看不出时**：改走盲区支线，禁止硬编「配料第一名是××」。  
5. 审核友好：观察与算账，不诊断、不开方。简体中文。
