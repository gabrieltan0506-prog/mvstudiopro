# 四季畅销品痛点槽点科普 Skill（弧 + 运镜）

## 用途
选题挂在**当季可感知的畅销消费品**上时启用。  
**不是轻飘飘科普**——要有共鸣痛点、槽点打脸、具体数字钉子；观众看完觉得「被说到了」，不是听了一堂淡课。

本 Skill = **叙事弧 + 画面语法**（含配料打脸主路径；细节句式仍可叠 `label-debunk-copy`）。  
权威一句见 `authority-cite-endorsement`（**正文必须落句**）；变现见 `fmcg-popsci-monetize`。

参考样本：`~/Downloads/2026Jul12/ice.mp4`（老洋人科普）；2026-07-13 再抽帧见 `~/Downloads/2026Jul13/ice-frames-dense/`。

## ice.mp4 / 雪糕金牌结构（可迁移 · 优先于「念机制」）
观众一看就懂的结构（创作者原话对齐）：
1. **一眼懂**：原来不能给孩子一直吃雪糕。  
2. **痛点1**：配料表原来有猫腻。  
3. **痛点2**：孩子食欲下降、影响发育。  
4. **会选了**：以后懂的选雪糕了。

**禁止**写成：「儿童添加糖与发育风险的循证综述」——那是论文腔，没人想看。

## ice.mp4 密度课（再抽帧结论 · 可迁移）
1. **日常赌注钩**：如果每天都…（一口就能代入）。  
2. **分屏/对比钉子**：成人 vs 孩子、以为 vs 其实——字幕短句砸槽点（「造成的伤害其实完全不一样」）。  
3. **最直接麻烦先抛**：如「最直接的麻烦就是糖」——痛点不绕弯。  
4. **机制画面具象**：糖/节律/反弹用可视符号，不念论文。  
5. **字幕节奏**：一句一个钉子，连续打脸，禁止轻描淡写「注意健康」。  
6. **高收藏感**：信息可截图对照（量感、上限占比），不是氛围散文。

## 痛点 / 槽点硬门槛（自检不过就重写）
- [ ] 有没有「就是我」的生活窘境？  
- [ ] 有没有一句能截图的槽点/反常识？  
- [ ] 有没有克数或一日上限占比？  
- [ ] 有没有「按《中国居民膳食指南…》」类权威一句？  
- [ ] 有没有立刻能做的动作（翻标/对比/算账）？
- [ ] 看完能不能复述「会选了」——而不只是「研究说不好」？

## 四季畅销品菜单（每条只挂 1 个主品）

### 春
花茶/新茶礼盒、清火饮料、酸奶杯、轻食代餐、踏青零食。

### 夏

### 秋

### 冬

### 全年常青

## 样本叙事弧
1. 日常赌注钩  
2. 反驳预埋  
3. 翻包装 / 营养成分  
4. 排序法则或盲区支线（叠 label-debunk）  
5. 认知打脸（槽点字幕级）  
6. 工业/工艺 2–3 点到为止  
7. 宣称拆穿  
8. 量感钉子（克数 / 一日添加糖几分之几）  
9. 体感反常识  
10. 关系线可选（孩子/长辈礼盒）  
11. **权威一句必须写出全文**（禁止只在心里想）  
12. 总账收束：不划算

## 运镜语法（摘要）
- 开场：断裂/拉丝/咬入特写 + 短字幕槽点。  
- 翻标：背面占满；圈前 2–3 名或糖/钠。  
- 盲区：营养成分 + 一日上限手卡；或宣称正反面分屏。  
- 收束：货架回环 +「怎么算都不划算」。

## 硬约束
1. 必须挂具体畅销品。  
2. 禁止轻描淡写；必须有痛点+槽点+量感。  
3. 禁止深医学词墙。  
4. 标签不够用就走盲区，禁止编造配料第一名。  
5. 简体中文；审核友好。
