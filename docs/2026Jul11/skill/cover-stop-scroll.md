# 封面停滑与信息层级 Skill

## 用途
专管「第一眼」竖版信息流封面。出图 Prompt **必须**遵守。宁可少字、姿势过火，也不要「正襟危坐要开课」。

## 硬约束（文案层·少字）
1. 屏上主句 **`coverHeadline` 8–14 字**，只说一个缺口；**禁止**把长标题/整段 hook/论文式副标整屏印上。
2. **提亮规则**：主句里只允许 **2–6 个关键字**用鲜明强调色（黄/亮蓝择一）标出；其余字用白/浅色大字即可。**不要**为「说完整」把剩下大段解释也画上去——没人想看。
3. 副标可选，**≤12 字**一行；总共可见文案 **≤2 行**。无副标往往更好。
4. **禁止百科封面**：书名墙、多图标+多辅标栏、说明书信息图、履历小字。
5. 正例气质（调适后，勿照抄人脸）：小红书知识作者那种——**大字少、关键字亮色、构图干净**；再叠我们的动态姿势与人设道具。

## 硬约束（有人物·表情姿势·禁止坐姿复读）
信息流不会点「坐着上课」的正经半身像。有人物时：

1. **强烈禁止默认**：正面端坐办公桌/咖啡桌、双手搭桌、端杯正视、证件照微笑、冷脸讲课、对称「老师开场」。
2. **每条封面必须换姿势**（同批六条不得六张都坐着）：至少选一类  
   - **表情**：瞪大眼不信、错愕张嘴、坏笑、大笑、陶醉眯眼、吃瘪、突然醒悟、夸张怀疑、憋笑、认真到入迷……  
   - **动态**：跳起来、差点摔倒/失衡、探身递东西、回头、举手比划、蹲下、侧身大步、歪头半遮脸、转身后仰、靠墙滑坐瞬间……  
3. 允许「递物怼镜头」「侧身失衡」等互动构图；**不允许**六条都是「坐姿+看镜头」。
4. 时装质感可留，但**不得**用「挺拔从容冷淡大片」压死表情与动态。
5. 锁脸时：同一人，表情姿势仍须丰富。

## 小红书作者风 → 为我所用（调适，非照搬）
- **要学**：少字大标题、关键字亮色、干净留白、脸可读、可选 **最多 1 个**极小视觉证据点（勿图标墙）。
- **要改**：把「正坐讲课脸」换成**有戏的表情/动态**；道具跟本人设（典籍、器官隐喻物件、运动、爵士等）走，不要默认医疗白大褂/手术帽，除非人设明确需要且审核友好。
- **不要学**：病名大字恐吓、临床特写堆叠、满屏说明书。

## 反例（禁止复刻）
- 一整段「为什么一杯冰咖啡、一本满是批注的论文、耳机里的爵士萨克斯……」印满上半屏。
- 人物端坐书桌、正脸微笑、像公司培训海报。
- 六张封面同一坐姿换背景。

## 正例方向（勿照抄）
- 主句短：「会休息的人呢？」；只亮「会休息」。
- 人跳离椅子瞪大眼，空咖啡杯怼镜头；桌上《内经》与手机 9% 电量只作道具，不印长文。
- 平台母语：小红书偏情绪短句；B站偏知识反差短句；视频号偏一句人话。

## 与出图
只用 `coverHeadline`（+可选极短副标）上屏；**禁止**把选题全文/长 hook 当封面正文；有人物就写清**表情+姿势**抓人瞬间。
