# 封面停滑与信息层级 Skill

## 用途
专管「第一眼」：竖版信息流封面。导演手法管分镜；本 Skill 管封面文案层级、视觉气质，以及**有人物时的表情/姿势抓人度**。出图 Prompt 必须遵守。

## 硬约束（文案层）
1. **主句 `coverHeadline`：8–14 字**，只说**一个**信息缺口；禁止空词「为什么」单独做主句。
2. **副信息一行 ≤18 字**，不抢主句；可无副信息。
3. 封面总可见文案 **≤2 行**；禁止长段论述、履历堆砌、四条以上卖点清单。
4. **禁止百科封面**：书名墙、多图标+多辅标栏、说明书式信息图当封面。
5. 人物脸/上半身须可读；主句不压脸。

## 硬约束（视觉气质）
- 走向：**生活化、年轻化、健康化、高级感、有特色**。
- **禁止**默认暗沉、低照度、压抑严肃、葬礼感/审讯室感光影。
- 优先：自然日光、户外/生活场域、明快但克制的配色、一张鲜明强调色。
- 高级 ≠ 死板：可以时尚大片，但要有温度与点击欲。

## 硬约束（有人物时·表情与姿势）— 一眼抓人
信息流用户不会点「看起来要上课」的正经正面半身像。封面有人物时：

1. **禁止默认**：正面端坐/端站、双手自然下垂、职业证件照微笑、冷脸讲课、对称构图的「老师开场」站姿。
2. **必须选一种抓人瞬间**（随选题换，勿每条重复同一姿势）：
   - **表情**：大笑、错愕、陶醉眯眼、坏笑、夸张怀疑、嘴角压不住、吃瘪、认真到入迷、突然醒悟……
   - **姿势/动态**：跳跃、差点摔倒/重心失衡、探身、回头、举手比划、蹲下捡东西、端着托盘侧身大步、靠墙、歪头、手挡镜头半遮脸、转身瞬间……
3. **情绪要写在脸上和身体上**：让人 0.3 秒觉得「有事发生」，而不是「等你开讲」。
4. 时装高级感可以保留（衣着质感），但**不得用「挺拔从容冷淡大片」压死表情与动态**；封面优先停滑，其次才是编辑感。
5. 锁脸参考人像时：脸要像同一人，**表情/姿势仍须丰富**，勿因锁脸退化成证件照。
6. 无人物纯物件/场景封面：用强动态构图或意外物件关系抓人，仍禁止死板居中说明书。

## 反例（禁止复刻）
- 大字「为什么」+ 长句哲学论述 + 履历小字 + 四图标清单。
- 整屏字多到需要「读完才懂」——信息流用户不会点。
- 博士/专家/服务员等身份封面：人物正面微笑看镜头，像公司培训海报。

## 正例方向（勿照抄）
- 一句具体反差：「学历放越前，人越不信？」
- 生活物件钩子：「冰箱贴比证书更劝人休息」
- 人物抓人：服务员端着盘子差点绊一下却笑得很得意；专家陶醉闭眼听爵士；对话里一方错愕一方坏笑。
- 平台母语：小红书偏情绪/清单大字；B站偏知识反差短句；视频号偏一句人话。

## 与出图
生成封面时：主标题用该平台 `coverHeadline`（或主 hook 压缩版）；场景跟文案场域走；**有人物就写清表情+姿势抓人瞬间**；**不要**为「显得专业」而加图标栏，也**不要**为「显得专业」画成上课正脸。
