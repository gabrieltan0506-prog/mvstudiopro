# 封面停滑与信息层级 Skill

## 用途
专管「第一眼」竖版信息流封面。出图 Prompt **必须**遵守。宁可少字、姿势过火，也不要「正襟危坐要开课」。

## 硬约束（文案层·少字）
1. 屏上主句 **`coverHeadline` 8–14 字**，只说一个缺口；**禁止**把长标题/整段 hook/论文式副标整屏印上。
2. **提亮规则**：主句里只允许 **2–6 个关键字**用鲜明强调色（黄/亮蓝择一）标出；其余字用白/浅色大字即可。**不要**为「说完整」把剩下大段解释也画上去——没人想看。
3. 副标可选，**≤12 字**一行；总共可见文案 **≤2 行**。无副标往往更好。
4. **禁止百科封面**：书名墙、多图标+多辅标栏、说明书信息图、履历小字。
5. 正例气质（调适后，勿照抄人脸）：小红书知识作者那种——**大字少、关键字亮色、构图干净**；再叠我们的动态姿势与人设道具。
6. 屏内字一律**中国大陆简体中文**。

## 硬约束（有人物·表情姿势·禁止坐姿复读）
信息流不会点「坐着上课」的正经半身像。有人物时：

1. **强烈禁止默认**：正面端坐办公桌/咖啡桌、双手搭桌、端杯正视、证件照微笑、冷脸讲课、对称「老师开场」。
2. **每条封面必须换姿势**（同批六条不得六张都坐着）。为**黄金三秒停滑与完播**，表情姿势宜**多元且可适度夸张**——帅气运动瞬间、登顶哲理瞬间、喜剧错愕瞬间都合格，按选题气质轮换：  
   - **表情（多元）**：瞪大眼不信、错愕、坏笑、大笑、陶醉、专注杀气、登顶释然、发球前凝神、突然醒悟、自嘲捂脸、假装正经……  
   - **动态·生活/喜剧（可夸张）**：跳起来、差点摔倒/失衡、探身递东西怼镜头、回头甩头、歪头半遮脸、夸张摊手……  
   - **动态·帅气运动（高度推荐轮换）**：网球发球展体、挥拍随球、爬山登顶举手/拄杖远眺、游泳出水、冲刺起步、击球重心转换……  
3. **题材融合**：同一封面/选题可同时服务「心肺/运动负荷」与「人生往高处走」——画面用发球/登顶等真动作，文案钩子可心肺机制 × 人生哲理对照（勿变成干讲课）。  
4. 允许「递物怼镜头」「侧身失衡」「发球定格」等强构图；**不允许**六条都是「坐姿+看镜头」，也**不允许**六条都挤同一搞笑表情，更**不允许**六条全是冷脸大片。  
5. 时装质感可留，但**不得**用「挺拔从容冷淡大片」压死表情与动态；运动场景服装跟场域走。  
6. 锁脸时：同一人，表情姿势仍须丰富；为完播可写「比日常略夸张一档」的微表情，忌尴尬演技说明书。

## 硬约束（场景服饰·防穿帮）
**动作决定穿什么**——这是出图最高优先级之一，压过「随便套件大片外套」：
1. 网球/挥拍 → 网球运动装（短袖或网球裙裤 + 球鞋 + 球拍）；**禁止长大衣、厚外套、正装西服边发球**。
2. 游泳/出水 → 泳装或刚披上的防水运动外套；禁止全套正装湿身。
3. 爬山/登顶 → 冲锋衣/功能外套/登山鞋；可有风感，但仍是户外装备。
4. 书房/厅堂访谈 → 才可用西装、丝绸衬衫、克制外套。
5. 锁脸时：脸必须是参考人；**衣着仍跟本条场景**，不可为保大片感硬套冲突服装。

## 小红书作者风 → 为我所用（调适，非照搬）
- **要学**：少字大标题、关键字亮色、干净留白、脸可读、可选 **最多 1 个**极小视觉证据点（勿图标墙）。
- **要改**：把「正坐讲课脸」换成**有戏的表情/动态**（含帅气运动，不限搞笑）；道具跟人设（典籍、运动、爵士、旅行等）走，不要默认医疗白大褂/手术帽，除非人设明确需要且审核友好。
- **不要学**：病名大字恐吓、临床特写堆叠、满屏说明书。

## 反例（禁止复刻）
- 一整段「为什么一杯冰咖啡、一本满是批注的论文、耳机里的爵士萨克斯……」印满上半屏。
- 人物端坐书桌、正脸微笑、像公司培训海报。
- 六张封面同一坐姿换背景；或六张全是夸张搞笑、没有一张运动/登顶帅气瞬间。
- **穿高定外套打网球 / 穿礼服进泳池**等场景服饰穿帮。

## 正例方向（勿照抄）
- 主句短：「会休息的人呢？」；只亮「会休息」。
- 网球发球定格：运动装、躯干打开、眼神盯球——可讲心肺负荷，也可讲「往高处打」的人生节奏。
- 爬山登顶远眺：冲锋衣被风掀起——可讲海拔/呼吸，也可讲「人生往高处行走」。
- 人跳离椅子瞪大眼 + 空咖啡杯怼镜头（喜剧线仍可用，勿垄断全批）。
- 平台母语：小红书偏情绪短句；B站偏知识反差短句；视频号偏一句人话。

## 与出图
只用 `coverHeadline`（+可选极短副标）上屏；**禁止**把选题全文/长 hook 当封面正文；有人物就写清**表情+姿势+场景服饰**抓人瞬间（帅气或有戏均可，忌死板坐姿与穿帮外套）。
