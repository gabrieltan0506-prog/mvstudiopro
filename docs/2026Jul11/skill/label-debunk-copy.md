# 配料打脸文案法 Skill

> 主弧与槽点密度以 `4season-fmcg-popsci` 为准；本 Skill 专管**句式武器**与「配料看不出」分支。fmcg 赛道路由会两者同挂。

## 什么时候用
预包装消费品要揭营销幻想、需要手上动作（翻面/对比）时。

## 什么时候不用
无包装物件、器官拟人主线、现制无标签却假装翻配料。

## 叠用
弧：`4season-fmcg-popsci` → 本 Skill 选 3–6 句 → 权威一句 `authority-cite-endorsement`（**必须出现在正文**）→ CTA：`hook-solution-cta`。

## 配料「看不出」
能翻表用表；不够则：营养成分对照膳食指南上限 → 品类标准/命名逻辑 → 公开抽检主题 → 体感算账。**禁止编造排序**。

## 句式武器（择用）
- 日常赌注：如果每天都来一根/一杯…  
- 反驳预埋：先别急着反驳  
- 排序法则：配料从高到低  
- 以为 X 其实 Y  
- 宣称拆穿：正面大字 vs 背面小字  
- 量感钉子：××克 ≈ 一日添加糖的几分之几  
- 儿童/银发线：口味养成、给长辈买礼盒的心疼（不诊疗恐吓）
