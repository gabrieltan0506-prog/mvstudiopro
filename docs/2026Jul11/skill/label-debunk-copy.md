# 配料打脸文案法 Skill

## 什么时候用（触发条件）
满足任一即启用本 Skill（常与 `4season-fmcg-popsci` 同开）：
1. 选题主角是**预包装 / 有包装文字**的食品或季节性消费品，且要「揭营销幻想」。  
2. 标题或钩子依赖「以为高级/健康/有奶/0糖 → 其实…」。  
3. 需要观众**手上有动作**（翻面、对比两款、看每 100 克）。  
4. 决策智库/自定义扩写里出现「添加剂恐吓」风险——用本 Skill 改成**可验证打脸**，避免都市传说。

## 什么时候不用
- 纯器官拟人、关系 CTA、无包装物件 → 用 `crossover-popsci` / `hook-solution-cta`。  
- 已明确是现制现售、无标签（路边摊软冰等）→ **不要假装翻配料表**；改体感、工艺常识、对比预包装、或权威「添加糖上限」算账。  
- 只有「我觉得不健康」没有证据路径 → 先补判断路径，再写句子。

## 怎么用（叠用顺序）
1. **弧与画面**：`4season-fmcg-popsci`  
2. **本 Skill**：选 3–6 个句式武器，中段打脸  
3. **权威一句**：`authority-cite-endorsement`（食品栏）  
4. **开场火花 + 结尾 CTA**：`hook-solution-cta`；变现漏斗与禁带货见 `fmcg-popsci-monetize`。  

一条里不要 11 种句式全堆；前 3 秒最多 1 个预埋 + 1 个赌注。

## 配料表「看不出」怎么办？（必须先想这条）

**现实**：很多打脸**不能**只靠配料第一名。复合配料、香精统称、工艺与油种、宣称话术、份量把戏，都会让「翻一下就赢」失效。  
**原则**：能翻表用表；表不够就换**第二证据**；第二证据也没有就改选题角度，**禁止编造配料排序**。

### A. 配料表仍够用的情况
- 前几位是水、糖、植物油、乳粉 vs 生乳等，观众默认信念可被直接打脸。  
- 写法：排序法则 + 以为 X 其实 Y。

### B. 配料表含糊 / 看不出时的替代证据（按优先级）

| 盲区 | 你怎么判断 | 口播怎么说 | 画面 |
|------|------------|------------|------|
| 只写「食用香精」「复合调味料」 | 不装懂具体化学名；打「香气/味道是配方设计」 | 「表上看不出是哪种香，但说明奶香不必来自奶」 | 香精瓶/工业淋面示意，勿假实验室报告 |
| 正面宣称「0蔗糖 / 低脂 / 植物基」 | **营养成分表**：能量、糖/碳水、脂肪、钠；对比每 100g 与每份 | 「0蔗糖≠没甜；先看每 100 克」 | 正反面分屏 |
| 「有奶味」但分不清鲜奶等级 | 看是否标注生乳/灭菌乳添加量；品类名（如能否叫「牛乳冰淇淋」）；蛋白质与乳脂肪相关话术要谨慎、只复述包装与国标分类逻辑 | 「名字好听，先核对它能不能叫那种名」 | 标准类别/品名特写 |
| 植脂 vs 乳脂 | 配料里的植物油/代可可脂 + 品类分级常识（全乳脂/半乳脂/植脂） | 「皮和酱常常更贵在样子」 | 脆皮掰开对比 |
| 份量把戏 | 营养成分「每份」很小；换算到一根/一袋/一餐 | 「按你真实会吃完的量算」 | 整支/整袋 vs 小份标注 |
| 现制无标签 | 不翻表；用体感、糖勺量感、对比预包装、权威添加糖上限 | 「摊上没有表，就用勺子跟指南比」 | 倒糖浆量感 |
| 添加剂吓人不成立 | 合法添加剂≠有毒；改打「用量角色」或「整体糖盐油」 | 「别背添加剂黑名单，先看糖盐油」 | 营养成分表钠/糖 |
| 需要外部证据 | 市场监管抽检通告、国标类别、膳食指南/WHO 上限（见权威 Skill） | 「指南里成年人添加糖最好…」 | 信息卡，勿伪造公文 |

### C. 仍禁止的写法
- 配料模糊却言之凿凿「第一名一定是××」。  
- 把合法食品添加剂说成毒药。  
- 用单篇自媒体检测当唯一真理。  
- 对观众下诊断（「你已经…」）。

## 句式武器库（按段落选用）

### 1）反驳预埋（黄金三秒）
「先别急着反驳。」——预判杠精，不骂观众蠢。

### 2）翻标签邀请
「嘿，你看配料表行吗？」「背面那行小字。」——抽象变手上动作。

### 3）排序法则
「配料是从高到低排的。」——后面打脸才站得住。

### 4）以为 X，其实 Y
「你以为是奶，前面往往是水和糖。」公式：`默认信念 → 可验证事实`。

### 5）工业分工三短句
乳化管质地、增稠管稠度、香精管「像奶」；每条最多 3 个角色。

### 6）调包点名
「脆皮？植物油混着代可可脂。」只点一件「看起来高级」的部件。

### 7）量感数字
「动辄十几克糖」；优先克数或「占指南一日上限的一半」。

### 8）体感反常识
「吃完可能更热」「越喝越渴」——按季节换，不限夏天。

### 9）儿童/银发加速器（慎用）
只打习惯、牙齿、口味、乱买礼盒；不做疾病恐吓。

### 10）权威一句（详单见权威 Skill）
「按《中国居民膳食指南》…」「世卫组织建议游离糖…」——**一句够**。

### 11）盲区坦白（新增）
「这张表看不出来细香精；但营养成分表的糖和钠，骗不了人。」——诚实比装懂更赢信任。

### 12）总账收束
「这笔账怎么算都不划算。」「钱进了包装和广告口袋。」

## 标题/封面主句偏好
- `如果每天都____` / `先看配料表` / `0蔗糖≠____` / `吃完更热`  
- 8–14 字；数字优先克数、名次、每 100 克。

## 硬约束
1. 打脸对物不对人。  
2. 半成品：至少留下「对比两款」或「先看每 100 克糖/钠」。  
3. 与 `4season-fmcg-popsci`、`authority-cite-endorsement`、`hook-solution-cta` 叠用边界如上。  
4. 简体中文；禁教学腔「首先其次综上所述」。
