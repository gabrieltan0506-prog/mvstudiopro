# 生活场域反坍缩 Skill

## 用途
人设含兴趣爱好、运动、旅行、艺术生活、学者审美时启用。  
解决两类坍缩：①「不是网球就是游泳」；②「不是书房古籍就是黑胶爵士 + 哈佛实验室旁白」。

## 硬约束（本批选题）
1. 若人设同时有多项兴趣，**禁止**六条选题场景几乎全是同一两项（如只打网球+游泳）。
2. 同一批中，单一运动/爱好作为主场景合计 **≤2 条**；其余须换场域。
3. **书房 / 古籍 / 黑胶爵士 / 精密实验室** 作为主视觉合计 **≤2 条**（四者算同一「学者橱窗」族）。其余必须落在烟火、关系、通勤、热播对照、户外等。
4. 场景须**具体可拍**（地点+物件+光线），禁止空喊「热爱生活」。
5. 同批至少覆盖 **3 类**场域；其中至少 **1 条**纯生活烟火（菜市场/路边摊/冰箱/通勤/亲子桌），至少 **1 条**热播或关系冲突（若人设允许）。

## 场域池（按人设有则优先，无则从池中合理拓展）
- **日常与关系（优先补齐）**：厨房冰箱、菜市场、宵夜摊、通勤地铁、亲子餐桌、银发陪伴、加班回消息、会议室散场。
- **运动与身体**：网球、游泳、慢跑、瑜伽、徒步、骑行、力量训练、太极——轮换，勿锁死前两项。
- **旅行与地理**：机场候机、古城巷弄、海边晨光、高原、温泉、火车窗景。
- **艺术与听觉（配额内）**：爵士现场、黑胶、音乐会、博物馆、画展、书法茶席。
- **知识与工作（配额内）**：实验室美学、讲台、书房窗光、诊所外等候区（审核友好，少临床强锚点）。

## 写法
- 用场域服务痛点与钩子，不要为晒爱好而晒爱好。
- 古今对照时，现代半边也要换场域，避免「古人 × 永远书房黑胶」。
- 医学名词可以出现，但**画面先给人的一天**，再点机制；禁止整条选题像论文摘要配书房 B-roll。

## 合格批次自检
- [ ] 本批主场景是否至少覆盖 3 类场域？
- [ ] 网球/游泳是否未超过 2 条？
- [ ] 书房/古籍/爵士/实验室族是否未超过 2 条？
- [ ] 是否有烟火日常 +（可选）热播/关系切口？
- [ ] 每条是否仍回扣客户痛点与 CTA？
