# 生活场域反坍缩 Skill

## 用途
人设含兴趣爱好、运动、旅行、艺术生活时启用。解决「不是网球就是游泳」的场景疲劳。

## 硬约束（本批选题）
1. 若人设同时有多项兴趣，**禁止**六条选题场景几乎全是同一两项（如只打网球+游泳）。
2. 同一批中，单一运动/爱好作为主场景合计 **≤2 条**；其余须换场域。
3. 场景须**具体可拍**（地点+物件+光线），禁止空喊「热爱生活」。

## 场域池（按人设有则优先，无则从池中合理拓展）
- **运动与身体**：网球、游泳、慢跑、瑜伽、徒步、骑行、力量训练、太极——轮换，勿锁死前两项。
- **艺术与听觉**：爵士现场、黑胶、音乐会、博物馆、画展、书法茶席。
- **旅行与地理**：机场候机、古城巷弄、海边晨光、高原、温泉、火车窗景。
- **日常与关系**：厨房冰箱、菜市场、书房窗光、银发陪伴、亲子餐桌、职场通勤。
- **知识与工作**：实验室美学、讲台、书房古籍、诊所外等候区（审核友好，少临床强锚点）。

## 写法
- 用场域服务痛点与钩子，不要为晒爱好而晒爱好。
- 古今对照时，现代半边也要换场域，避免「古人 × 永远网球场」。

## 合格批次自检
- [ ] 本批主场景是否至少覆盖 3 类场域？
- [ ] 网球/游泳是否未超过 2 条？
- [ ] 每条是否仍回扣客户痛点与 CTA？
