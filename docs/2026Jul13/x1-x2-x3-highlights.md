## x1 搜索瀑布流封面（親子旅遊清單）可迁移
1. **杀伤词**：必備 / 清單 / 帶娃 / 保姆級 / 不踩雷 / 救命 / 建議收藏 / 懶人必看  
2. **证据画面**：摊开行李箱、收纳分区、药盒零食特写、手写/备忘录清单怼镜头、机舱/酒店真实场景  
3. **大字少行**：竖版封面主句短、对比强（黄/白描边）；一眼知道「存了有用」  
4. **反推荐也香**：「别买这些」「后悔带的」比空泛好物墙更像真话  
5. **年龄/场景钉**：几岁娃、出国 vs 国内、几小时大交通——越具体越好搜

## x2 出行妙招（樱桃小樱子）可迁移
1. 标题：**省钱省事 + 必备小妙招 + 你一定用得上**  
2. 章节条：高效打包 → 收纳瓶罐刷具 → 酒店卫生 → 酒店安全（可跳播）  
3. 画面：行李箱 POV 叠衣、**5 毛离心管**收刷具、保鲜膜包鞋等「0 成本」钉子  
4. 字幕一句一钉；收束感谢关注——工具帖气质  
5. 结构：多个小妙招串联，**不是**一篇旅游论文

## x3 带娃 12 神器（小熊是嘻嘻麻麻）可迁移
1. 封面：**一口气分享 12 个** + flat lay 神器墙 + **被问爆 / 无广**  
2. 顶栏进度：`1|2|…|12` + 当前序号大圆标（如 02 便携饮水机 / 04 饭盒袋 / 05 蒙氏安静书）  
3. 每件钉 **结果句**：矿泉水瓶插上去；✅玩水打湿的衣服；**全程安安静静**  
4. 真场景演示：火车桌、沙滩、酒店——禁棚拍说明书墙  
5. 藏 6 万 ≫ 赞 3.4 万 → 清单可截、可复购决策

## 生成侧必须对齐（封面 + 笔记）
- 清单/攻略向封面：`大数字件数` + `场景证据` + `结果钉`（安静/不踩雷/少带/省事）  
- 禁止：读论文式主句、正坐讲课脸、无证据的空鸡汤大字  
- 正文/分镜：雪糕公式（一眼懂 → 痛点猫腻 → 后果 → 会选）叠清单墙  
- 蓝海词如「親子旅遊清單」进钩子/发布建议，**不做**文末标签墙
