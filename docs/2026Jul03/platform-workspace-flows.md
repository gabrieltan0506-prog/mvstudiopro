# /platform · 自定义创作工作台 · 用户流程

> 对应 PR #694 最终 UX。每 Tab 均为**同页完成**，任务进行中下方全案分析区自动收起。

---

## Tab 1 · 自定义文案

### 步骤说明（UI 卡片）

| 步 | 标题 | 说明 |
|----|------|------|
| 1 | 粘贴文案 | 贴入 Markdown 或分镜脚本，并选择生成类型。 / 可选「优化自定义文案」先深度改写再出图。 |
| 2 | 生成结果 | 点击生成按钮，等待任务完成。 / 图片或优化稿直接显示在本 Tab 下方。 |

### 操作流

1. 选择类型：**单页图文卡片**（50 cr）/ **2×4 分镜**（60 cr）/ **优化自定义文案**（25 cr）
2. 粘贴文案（优化模式可填「优化要求」）
3. 点击生成
4. 优化模式：结果区可「用优化稿生成分镜 / 图文卡片」
5. 可选：**导出 PDF**（Tab 栏旁按钮，#691）

## Tab 2 · 自定义选题

### 步骤说明

| 步 | 标题 | 说明 |
|----|------|------|
| 1 | 填写设定 | 写主人公特质与专长，可选填选题标题。 / 勾选需要生成的文案、封面或分镜。 |
| 2 | 上传人像 | 上传参考人像，封面与分镜会融合相貌。 / 生成封面/分镜时必填；仅文案可跳过。 |
| 3 | 一键生成 | 文案扩写首次免费，图片积分见按钮。 / 结果在本 Tab 预览，可下载或设为参考。 |

### 操作流

1. 勾选：文案 / 封面 / 分镜（可多选）
2. 填主人公特质（文案必选时必填）
3. 选 2×4 或 3×4 分镜网格
4. 上传人像（封面/分镜需要时）
5. 点击生成 → 预览文案卡片、封面、分镜

## Tab 3 · 素材分析（成长营迁移主链路）

### 步骤说明

| 步 | 标题 | 说明 |
|----|------|------|
| 1 | 上传并分析 | 添加封面或分镜 PNG/JPG。 / 点击「开始视觉分析」，结果只绑定你的素材。 |
| 2 | 深度优化 | 基于分析改写封面、分镜与发布稿。 / 消耗 25 积分，引用近期热点、不用旧套话。 |
| 3 | 生成图片 | 用优化稿直接出分镜或图文卡片。 / 分镜 60 积分 · 卡片 50 积分。 |

### 操作流

1. （可选）填业务背景
2. 添加 PNG/JPG，点击「开始视觉分析」（40 cr/张）
3. 查看视觉分析结果 → 点击「步骤 2 · 深度优化」
4. 查看优化稿 → 点击「步骤 3 · 生成分镜」或「生成图文卡片」
5. 生成图出现在本 Tab 下方（上篇/下篇）

**禁止路径**：不调用 growth mock snapshot；热点来自 trendStore 近 7 天 live 样本。

## Tab 4 · 自定义抠像

### 步骤说明

| 步 | 标题 | 说明 |
|----|------|------|
| 1 | 描述场景 | 写人物姿态、服装与背景提示词。 / 需要白底主体请在描述中注明「去背景」。 |
| 2 | 生成下载 | 选择比例与张数，点击开始生成。 / 单张按 plans 定价，2/4 张有折扣。 |

### 操作流

1. 输入主体描述
2. 选画面比例（9:16 / 16:9 / …）
3. 选 1 / 2 / 4 张
4. 生成 → 预览 → 下载或「设为参考人像」（跳转选题 Tab）

## 全局行为

| 行为 | 说明 |
|------|------|
| busy 收起 | 任一 Tab 任务进行中 → 下方「平台顾问台 / 全案分析」隐藏 |
| Tab 互锁 | busy 时其他 Tab 按钮 disabled |
| 旧成长营 | `/creator-growth-camp` → `/platform` |
| Legacy | `/creator-growth-camp/legacy` 保留旧页 |
