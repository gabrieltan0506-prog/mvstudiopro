## 澎湃/DataEye 百强榜扫描（2026-07-15 文）

来源：[澎湃转载 DataEye 百强](https://www.thepaper.cn/newsDetail_forward_33588420)（湃客转载，数字以原文为准）。

### 抖音端原生百强（要点）

- 上榜作品单月播放增量合计约 **591.31 亿**；TOP6 漫剧增量超 10 亿。
- 榜首：3D 漫《发配边关，罪妻开荒养出战神》约 **21.27 亿**（6 月上线，断层第一）。
- 第二：AI 仿真人剧《财神天降，送我上青云》（灵犀影业）约 12.74 亿；第三：3D 漫《渔乡守真心》约 11.80 亿。
- **类型结构**：AI 仿真人剧 **76** / 3D 漫 **16** / 2D **4** / 沙雕 **3** / 解说 **1**。
- **题材内核**：下沉爽文 + **逆袭**（废柴/重生/打脸虐渣/种田致富）；仿真人侧偏都市日常、家庭伦理、年代怀旧、系统流、赘婿流。
- 制作方提及：灵犀影业、铭兴文化、硕才、新漫跳动、宇瀚忠毅、西垚数字等。

### 红果百强（要点）

- TOP10 **全是 3D 漫**；上榜峰值热度总额约 60.54 亿量级（原文口径）。
- 热度 TOP3：`万妖图录传第七季`（约 9442 万）> `发配边关…第六季`（约 8251 万）> `万妖图录传第六季`（约 8068 万）。
- **季播/IP 连载**是主流打法：`万妖图录传`、`发配边关`（全系列多季进百强）、`聚宝仙盆之杂灵根才是真BOSS` 等多季占榜。
- **类型结构**：AI 仿真人 **58** / 3D 漫 **39** / 2D **3**（3D 热度占比显著高于抖音结构）。
- **题材**：架空玄幻、脑洞、仙侠逆袭为主，与抖音「下沉现实」对照鲜明；3D 适合宏大修仙/斗法场景。
- 出品方提及：大多多动画、陈柯文化（各 6 部量级）、光宇、爱看互动、灵矩动漫；仿真人侧如时刻互动《风水天师》等。

### 对我们工厂的含义

- 做 **3D/漫剧向** 内容：红果榜更对齐（玄幻世界观 + 季播钩子）；抖音榜仿真人占比更高，偏生活质感爽文。
- 选题优先：**逆袭钩子 + 可续季的世界观**；边关种田、妖界图鉴、杂灵根逆袭等标签可进题材关键词试验。

## 钛媒体：国内制作 vs 出海题材（2026-02-26）

来源：[AI漫剧2026：千亿深水区的工业化豪赌与审美突围](https://www.tmtpost.com/7890548.html)（吴怼怼 / 钛媒体；数字以原文为准）。

### 文中与「题材/市场」直接相关的点

- **国内平台流量池**：抖音、红果是主战场；文中举例《斩仙台下，我震惊了诸神》等在红果/抖音换播放；题材侧常见 **仙侠震惊流、IP 吞噬进化、末世废柴逆袭** 等国产爽文模板。
- **IP 路径**：阅文/番茄开放海量网文；近半年爆款中 IP 改编占比称超 60%；《从赖皮蛇开始吞噬进化》作「IP+AI 产能」案例。
- **出海专节（§07）**：
  - 平台：DramaBox（点众）、ReelShort（枫叶）、DramaWave（昆仑万维）等，千万级海外月活口径。
  - **题材策略原文要点**：用 AI 快速生产符合**西方受众**口味的「**狼人、吸血鬼、系统觉醒**」，**而非强行输出国漫 IP**。
  - 优势：漫剧比真人剧文化门槛低，AI 做本土化改皮成本更低；打法是「数据驱动 + AI 产能」。

### 题材对照表（国内制作 vs 出海）

| | 中国制作（结合钛媒体 + DataEye 百强） | 出海制作 |
|--|--------------------------------------|----------|
| 情绪内核 | 共通：**逆袭 / 打脸 / 升级** | 共通：同样要爽，但皮相本地化 |
| 高频皮相 | 古言边关种田、仙侠脑洞、妖界图鉴、末世开超市/当老板类模板、仿真人都市伦理/赘婿系统 | **狼人、吸血鬼、系统觉醒**；英语面孔的「废物老公其实是大佬」类逆袭（见钛媒体姐妹篇补充） |
| 世界观 | 红果偏架空玄幻可季播；抖音偏下沉现实可代入 | 西幻超自然 + 短剧节奏；忌原装国漫符号硬推 |
| 产能逻辑 | 抽卡师/提示词 + 角色一致性；成本卷到每分钟千元级 | 同一流水线，**换设定库与面孔库**做多地区版本 |
| 合规/纠纷 | 模板撞车诉讼（末世废柴类）、平台侵权快反；国内审核/AI 标识另线 | 版权与平台规则随目标国；文化冒犯风险（宗教/种族刻板）需自检 |
| 对我们 | 国内产品：题材词走边塞/修仙/末日/都市逆袭 | 若做出海线：工厂需另备 **狼人/吸血鬼/系统** 场景与人设锚点，而不是把「发配边关」直译出去 |

### 一句话结论

- **中国做**：先选平台口味（抖音仿真人下沉 vs 红果 3D 玄幻），题材钉在「逆袭 + 可识别国风/网文类型」。  
- **出海做**：**保留爽点骨架，替换文化皮**（狼人/吸血鬼/系统觉醒等），用 AI 本地化，不把国漫 IP 原装硬塞。

## 南方都市报：爆款与合规「成人礼」（2026-04-29）

来源：[月增700亿播放！AI漫剧的爆款与合规“成人礼”](https://www.oeeee.com/html/202604/29/1702585.html)（南方都市报 / 奥一网）。

### 市场与爆款（文中点名）

- DataEye：2026 年 3 月播放增量破 **700 亿**；新上线漫剧约 4.7 万部。
- 红果：`《菩提临世真人AI版》` 上线不到一周热度约 8900 万，文称 **AI 剧首次登顶总榜、超越真人短剧**。
- 现象级节奏：`《气运三角洲，我凭操作吊打全球》`（改编游戏《三角洲行动》）**3 人 / 5 天**，上线 **29 小时破 2 亿**；另有《斩仙台下…》全网超 10 亿、《西游，错把玉帝当亲爹》35 小时破亿等。
- **女频/甜宠**跑出黑马：古董夫君甜宠、公主超能力都市异能等；受众仍偏年轻男性基本盘，女频潜力未完全释放。
- **游戏厂入局**：游戏 IP → 漫剧宣发/破圈；三七「灵梦」等工作流工具被提及。

### 合规（对国内上架极关键）

- **2026-04-01 起**：广电要求 AI 漫剧 **先备案后上线**；未备案全网下架；纳入网络微短剧分类分层审核（「投资额+题材」三级）。
- 合规三维度（文中归纳）：**内容**（暴恐黄、歪曲历史、未成年更严）、**版权**（训练/素材/杂糅剧本仍可能侵权）、**技术申报**（如实申报 AI 生成）。
- 红果一季度公告口径：已下架违规漫剧 **1718** 部。
- 高发侵权：角色显著特征撞知名 IP、配音素材、多部小说情节喂 AI 杂糅。
- 反面案例：末世开超市类抄袭下架；「8 小时生 999 个娃」类低俗恶评。

### 与题材策略的关系

| 方向 | 南都文补充 |
|------|------------|
| 男频爽文 | 仙侠震惊流、游戏改编操作碾压、西游恶搞、团宠铁汉爹等仍在抖音热播 |
| 女频 | 甜宠、都市异能、古董夫君向可做第二曲线 |
| 国内上架 | **备案/版权/底线** 比「能不能做出片」更先决定生死 |
| 工厂含义 | 选题可试「游戏 IP 改编爽文」「女频甜宠」关键词；上架路径必须预留备案与版权筛查，不鼓励堆低质擦边 |
